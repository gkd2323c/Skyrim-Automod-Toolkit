This may affect not only the guard you're talking to, but also every guard within your cell. This is probably a bug with the way that phrases are randomly generated.

The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim:Mod_Unofficial_Skyrim_Patch), version 1.2.2, fixes this bug.
