[![](https://images.uesp.net/thumb/a/a8/SR-item-Shrouded_Gloves.jpg/200px-SR-item-Shrouded_Gloves.jpg)](https://en.uesp.net/wiki/File:SR-item-Shrouded_Gloves.jpg) [](https://en.uesp.net/wiki/File:SR-item-Shrouded_Gloves.jpg) Shrouded Gloves Shrouded Gloves serve as the gauntlets for a set of armor used by the [Dark Brotherhood](Skyrim_Dark_Brotherhood.md). Alongside these gloves, this set of armor also includes [armor](https://en.uesp.net/wiki/Skyrim:Shrouded_Armor), [boots](https://en.uesp.net/wiki/Skyrim:Shrouded_Boots), and a [cowl](https://en.uesp.net/wiki/Skyrim:Shrouded_Cowl). These gloves cause [backstabs](https://en.uesp.net/wiki/Skyrim:Backstab) to do twice the normal damage, making these lethal sneak attacks even more effective. [Tempering](Skyrim_Workbench.md) Shrouded Gloves requires a piece of [leather](Skyrim_Leather.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.
