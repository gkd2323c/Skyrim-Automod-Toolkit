# Specialty Gear

| --- | --- |

This page summarizes all of the **Specialty Gear** in Skyrim. These are items that can be found in many locations in the game, yet do not qualify as generic weapons or generic armor. For example, it includes weapons and armor that are only used by specific races or factions (e.g., the Stormcloaks, the Thieves Guild, the Blades). It also includes some items, such as pickaxes and woodcutter's axes, that are only used in certain situations.

- Oddly enough, the Blades Sword has an associated perk (Steel), whereas the Blades Armor set does not. This seems inconsistent, given that all of the items need a Steel Ingot to temper them.
