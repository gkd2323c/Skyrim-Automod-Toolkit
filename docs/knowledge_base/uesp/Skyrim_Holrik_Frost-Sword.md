[![](https://images.uesp.net/thumb/2/2c/SR-npc-Holrik_Frost-Sword.jpg/200px-SR-npc-Holrik_Frost-Sword.jpg)](https://en.uesp.net/wiki/File:SR-npc-Holrik_Frost-Sword.jpg) [](https://en.uesp.net/wiki/File:SR-npc-Holrik_Frost-Sword.jpg) Holrik Frost-Sword's ghost **Holrik Frost-Sword** is a [ghost](Skyrim_Ghost.md) found in [Cronvangr Broodlair](https://en.uesp.net/wiki/Skyrim:Cronvangr_Broodlair). He wears [Banded Iron Armor](Skyrim_Banded_Iron_Armor.md), [Iron Gauntlets](Skyrim_Iron_Gauntlets.md), [Iron Boots](Skyrim_Iron_Boots.md) and an [Iron Helmet](Skyrim_Iron_Helmet.md). He is equipped with the [Ice Blade of the Monarch](Skyrim_Ice_Blade_of_the_Monarch.md). He is involved in the [Caught in a Web](Skyrim_Caught_in_a_Web.md) quest.

## Related Quests
- **[Caught in a Web](Skyrim_Caught_in_a_Web.md)**<sup>[CC](Skyrim_The_Contest.md)</sup>: Search for legendary artifacts in a spider den.

## Notes
- There are in fact two versions of Holrik Frost-Sword: - The ghost found in Cronvangr Broodlair has a base ID of `FExxx80D` and a ref ID of `FExxx811`.
- His body is actually considered to be a container in game data, and has a base ID of `FExxx809` and a ref ID of `FExxx80C`. The [Ice Blade of the Monarch](Skyrim_Ice_Blade_of_the_Monarch.md) can be found on the body.
