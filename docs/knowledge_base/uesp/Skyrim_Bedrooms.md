# Bedrooms


The Bedrooms, like its counterparts, the [Armory](Skyrim_Armory.md) and [Storage Room](Skyrim_Storage_Room.md), has a large patio area on its roof when completed. This patio can interconnect with the Storage Room's patio if you build it onto your home as well. A few varieties of furniture, as well as lighting, can be added to this exterior area.

The following is a list of possible furniture items and decorations which can be added by using the carpenter's workbench in the Bedrooms addition. It is not necessary to purchase or build all furnishing options; you can pick and choose to better dictate style. However, all furnishing options are in a prearranged location and only ingredients and standard items can be moved. Oftentimes, the name of the furnishing option only indicates the primary piece of furniture, and actually includes various additional [containers](Skyrim_Containers.md) and accessories.

## Notes
- If you build the Bedrooms addition, you must build 2 Child's Beds and 2 Child's Chests in order to [adopt](Skyrim_Adoption.md) children; the Single Beds in the [Main Hall](https://en.uesp.net/wiki/Skyrim:Main_Hall) are not sufficient.
- If the family occupies the beds in the Bedrooms and the Main Hall's second level is also furnished with beds, the Manor's Steward and resident Housecarl will use the vacant beds in their scheduled sleeping hours, assuming they're not your spouse (and if the steward IS the housecarl, they obviously only need the one bed: if they are separate people but only one bed is constructed, one of them will have to sleep somewhere else.)
