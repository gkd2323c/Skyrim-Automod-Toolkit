[![](https://images.uesp.net/thumb/e/e0/SR-npc-Fort_Commander_02.jpg/200px-SR-npc-Fort_Commander_02.jpg)](https://en.uesp.net/wiki/File:SR-npc-Fort_Commander_02.jpg) [](https://en.uesp.net/wiki/File:SR-npc-Fort_Commander_02.jpg) A Fort Commander The **Fort Commander** is available once a side in the [Civil War](Skyrim_Civil_War.md) has taken over a fort. If there is a Fort Commander's Quarters he will be found in them, if not he can be found inside the fort. Apart from his name he is a generic [Imperial soldier](Skyrim_Imperial_Soldier.md) and has generic [Imperial soldier](Skyrim_Imperial_Soldier.md) dialogue.
