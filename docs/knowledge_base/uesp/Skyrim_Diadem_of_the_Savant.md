[![](https://images.uesp.net/thumb/9/9d/SR-item-Diadem_of_the_Savant.jpg/200px-SR-item-Diadem_of_the_Savant.jpg)](https://en.uesp.net/wiki/File:SR-item-Diadem_of_the_Savant.jpg) [](https://en.uesp.net/wiki/File:SR-item-Diadem_of_the_Savant.jpg) The Diadem of the Savant The Diadem of the Savant is a unique [circlet](Skyrim_Circlet.md) that serves as the reward for conquering [Shalidor's Maze](Skyrim_Shalidor%27s_Maze.md), although a respawnable version can also be found outside of [Froki's Shack](Skyrim_Froki%27s_Shack.md) (see [bugs](Skyrim_Diadem_of_the_Savant.md#Bugs)). As would be expected for an item associated with [Shalidor](Lore_Shalidor.md), the circlet makes it so that all spells use less magicka. While it may be useful to mages due to its enchantment, and since it can be worn without disabling the [Mage Armor](Skyrim_Mage_Armor.md) perk, its defensive capabilities cannot be improved further as it cannot be [tempered](Skyrim_Workbench.md).

## Notes
- This is technically Light Armor, but because it does not have the `Armor Light` keyword, you can wear it without disabling the [Mage Armor](Skyrim_Mage_Armor.md) perk. It will count as valid helmet for either the [Custom Fit](Skyrim_Custom_Fit.md) or [Well Fitted](https://en.uesp.net/wiki/Skyrim:Well_Fitted) perk.
- Using this item and the [Fortify Restoration](Skyrim_Fortify_Restoration.md) trick together can make all spells cost 0 magicka while occupying only one armor slot.
- This item is classified as light armor when trying to sell it; you can only sell it to general goods merchants and blacksmiths unless the Merchant perk is active.
- This item cannot be disenchanted at an Arcane Enchanter.

## Bugs
- This item cannot be improved at a workbench. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.1, addresses this issue. It now uses a [Gold Ingot](Skyrim_Gold_Ingot.md) for tempering.
- On the wood chopping block outside to the north-northwest of Froki's Shack, you can find a Diadem of the Savant, which respawns. It is possible this was a mistake, as the Creation Kit specifies that the diadem was meant to be a reward for completing Shalidor's Maze. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.2.5, addresses this issue. It changes this instance of the diadem into a leveled circlet with a [Fortify Archery](Skyrim_Generic_Magic_Apparel.md#Fortify_Archery) effect.
- It doesn't have the `Armor Jewelry` keyword set like other circlets. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.1.1, fixes this bug.
- It is possible for the Dremora holding the Diadem of the Savant to fail to be teleported back out of Shalidor's Maze when you solve it. If this happens, the Diadem is unobtainable. - The [Unofficial Skyrim Special Edition Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Special_Edition_Patch), version 4.2.9, addresses this issue. A copy will now be generated should this fail to work as expected.

- Another copy of the Diadem of the Savant an also be found on the wood chopping block outside Froki's Shack, provided you don't have Unofficial Skyrim Patch 1.2.5 or later.
