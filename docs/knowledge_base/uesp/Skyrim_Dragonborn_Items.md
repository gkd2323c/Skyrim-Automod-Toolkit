# Dragonborn Items

| --- |

Dragonborn adds a vast number of additional **items**, rewards, and new oddities to be discovered or wielded. More detailed information can be found about these items, how they work, or how to obtain them using this page as a guide.

## Alchemy

### Ingredients
- **[Ash Creep Cluster](https://en.uesp.net/wiki/Skyrim:Ash_Creep_Cluster)** — Abundant across [Solstheim](Skyrim_Solstheim.md).
- **[Ash Hopper Jelly](https://en.uesp.net/wiki/Skyrim:Ash_Hopper_Jelly)** — Dropped by [ash hoppers](Skyrim_Ash_Hopper.md).
- **[Ashen Grass Pod](https://en.uesp.net/wiki/Skyrim:Ashen_Grass_Pod)** — Abundant across Solstheim. 33% more potent [Resist Fire](Skyrim_Resist_Fire.md) effect.
- **[Boar Tusk](https://en.uesp.net/wiki/Skyrim:Boar_Tusk)** — Dropped by [bristlebacks](Skyrim_Bristleback.md) and [rieklings](Skyrim_Riekling.md). 5× longer lasting [Fortify Health](Skyrim_Fortify_Health.md) effect (only matched by [Giant's Toe](Skyrim_Giant%27s_Toe.md)). 25% more potent [Fortify Stamina](https://en.uesp.net/wiki/Skyrim:Fortify_Stamina) effect than any base game ingredient.
- **[Burnt Spriggan Wood](https://en.uesp.net/wiki/Skyrim:Burnt_Spriggan_Wood)** — Dropped by [burnt spriggans](Skyrim_Burnt_Spriggan.md).
- **[Emperor Parasol Moss](https://en.uesp.net/wiki/Skyrim:Emperor_Parasol_Moss)** — Found near [Tel Mithryn](Skyrim_Tel_Mithryn_(settlement).md). 50% more potent [Damage Health](Skyrim_Damage_Health.md) effect, on par with [Deathbell](https://en.uesp.net/wiki/Skyrim:Deathbell).
- **[Felsaad Tern Feathers](https://en.uesp.net/wiki/Skyrim:Felsaad_Tern_Feathers)** — Dropped by [Felsaad terns](https://en.uesp.net/wiki/Skyrim:Felsaad_Tern).
- **[Netch Jelly](https://en.uesp.net/wiki/Skyrim:Netch_Jelly)** — Dropped by [netch](Skyrim_Netch.md). 2× more potent [Restore Stamina](Skyrim_Restore_Stamina.md) effect (most potent of all ingredients, including later [Creation Club](Skyrim_Creation_Club.md) [additions](Skyrim_Ingredients.md#Creation_Club_Ingredients)). The only non-Creation Club ingredient with [Paralysis](Skyrim_Paralysis.md) as first effect (useful to nullify [fall damage](Skyrim_Damage.md#Fall_Damage)).
- **[Scathecraw](https://en.uesp.net/wiki/Skyrim:Scathecraw)** — Abundant across Solstheim.
- **[Spawn Ash](https://en.uesp.net/wiki/Skyrim:Spawn_Ash)** — Dropped by [ash spawn](Skyrim_Ash_Spawn.md).
- **[Trama Root](https://en.uesp.net/wiki/Skyrim:Trama_Root)** — Abundant across Solstheim.
