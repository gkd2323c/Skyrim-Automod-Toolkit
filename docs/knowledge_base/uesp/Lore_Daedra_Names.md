This is a list of all the known **[Daedra](Lore_Daedra.md) names**, compiled from the games.

*[Battlespire](https://en.uesp.net/wiki/Battlespire:Battlespire)* non-expositionally established that a Daedra's first name is usually their clan/race name, followed by a given name—see examples: the [Dremora](https://en.uesp.net/wiki/Battlespire:Dremora#Unique_Dremora), [Herne](https://en.uesp.net/wiki/Battlespire:Herne#Unique_Herne), [Perthan](https://en.uesp.net/wiki/Battlespire:Spider_Daedra), and [Xivilai](https://en.uesp.net/wiki/Battlespire:Xivilai_Moath). [ESO](https://en.uesp.net/wiki/Online:Online) continues this motif with the [Elemental Daedra](Lore_Elemental_Daedra.md) —see examples: the [Nomeg](Lore_Daedra_Names.md#Frost_Atronach), [Magdra](Lore_Daedra_Names.md#Flame_Atronach), and [Zymel](Lore_Daedra_Names.md#Storm_Atronach). Though not all Daedra follow this motif.
