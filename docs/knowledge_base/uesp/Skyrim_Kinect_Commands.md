| Dragon Shouts | | | | |
| *See [*Dragon Shouts*](Skyrim_Dragon_Shouts.md) for a list of both English and Dragon Language commands*. To activate the shouts or powers without using Kinect, press [](https://en.uesp.net/wiki/File:UESP-icon-Xbox_LB.png) + [](https://en.uesp.net/wiki/File:UESP-icon-Xbox_RB.png). While Kinect is disabled, use [](https://en.uesp.net/wiki/File:UESP-icon-Xbox_RB.png) in order to activate the shouts as normal. <br> <br> - In order to activate the shouts, they must be unlocked at the level you are trying to use. <br> - Shouts can be said in English or Dragon Language. <br> - English shouts can be used without holding [](https://en.uesp.net/wiki/File:UESP-icon-Xbox_RB.png), while Dragon Language shouts require that [](https://en.uesp.net/wiki/File:UESP-icon-Xbox_RB.png) be held (If you're having trouble casting the shouts, try to say them in a fluent pace at a fast speed). | | | | |

| Container Menu Only | | | | |
| SET LOOT LIMIT <value> | Sets the value that the LOOT ITEMS voice command should use. (Value = 0, 5, 10, 25, 50, 100) | | | |
| LOOT ITEMS | Takes all items from a container whose value-to-weight ratio is greater than or equal to the loot limit. For example, if the loot limit is 100, an object with a weight of .1 and value of 100, would have a value to weight ratio of 1000, so it would qualify. If the loot limit is 0, only weightless items with value are taken, i.e. gold. | | | |
