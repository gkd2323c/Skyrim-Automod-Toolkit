[![](https://images.uesp.net/thumb/5/5e/SR-item-Tumblerbane_Gloves.jpg/200px-SR-item-Tumblerbane_Gloves.jpg)](https://en.uesp.net/wiki/File:SR-item-Tumblerbane_Gloves.jpg) [](https://en.uesp.net/wiki/File:SR-item-Tumblerbane_Gloves.jpg) Tumblerbane Gloves Tumblerbane Gloves are a special version of [Shrouded Gloves](Skyrim_Shrouded_Gloves.md) given to you for refurbishing the [Dawnstar Sanctuary](Skyrim_Dawnstar_Sanctuary.md) with a master bedroom. The gloves can be found at the foot of the newly purchased bed. As the name suggests, these gloves have been enchanted to [fortify lockpicking](Skyrim_Fortify_Lockpicking.md) skills, making it twenty percent easier to break into chests.

## Bugs
- Tumblerbane Gloves cannot be improved at a [workbench](Skyrim_Workbench.md). - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, addresses this issue. It enables upgrading with [Leather](Skyrim_Leather.md) but uses no [Smithing](Skyrim_Smithing.md) perks.
