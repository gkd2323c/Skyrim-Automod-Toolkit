# Teleport Pet (effect)

| [](https://en.uesp.net/wiki/File:SR-icon-spell-Alteration.png) | Teleport Pet |
| --- | --- |
| School | [Alteration](Skyrim_Alteration.md) |
| Type | Other |

**Teleports <Creature>**

Teleport Pet allows the caster to teleport a specific [pet follower](Skyrim_Followers.md#Creation_Club) to their current location, all of which are added through [Creations](Skyrim_Creation_Club.md). A different magical effect is used for each different creature that can be teleported.

## Spells
The following spells use the Teleport Pet effects. Following each spell is the specific variant of the effect that it uses:

- [Teleport Pet: Arachnia](Skyrim_Teleport_Pet__Arachnia.md): *Teleport Arachnia* (`Form ID: FE [xxx](Skyrim_Form_ID.md#Creation_Club) 918`; `cc VSVSSE002_MGEF_Teleport Spider`)
