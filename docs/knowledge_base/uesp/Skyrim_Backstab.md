# Backstab

*Enchanting description:* **Double sneak attack damage with one-handed weapons.**
The magnitude of this effect is always 0 as an enchantment, despite the actual perk given being a hidden value. This value is under the label of 'Modify Sneak Attack Mult' and set to a value of 2 in all cases. The value of the enchantment has no effect on the magnitude, though no custom enchantments are available. It doubles the Sneak Attack Multiplier for any sneak attack with a one-handed weapon, including the 6× multiplier from the [Backstab](Skyrim_Sneak.md#Backstab) perk and the 15× multiplier from the [Assassin's Blade](Skyrim_Sneak.md#Assassin.27s_Blade) perk.

## Enchanting
The following items use this effect, but none of them can be disenchanted. They are only obtainable through the [Dark Brotherhood](Skyrim_Dark_Brotherhood.md) (except for the Shrouded Gloves and Shrouded Hand Wraps, which may be looted from [Astrid](Skyrim_Astrid.md) in [With Friends Like These...](Skyrim_With_Friends_Like_These....md) and the other members of the Dark Brotherhood in [Destroy the Dark Brotherhood!](Skyrim_Destroy_the_Dark_Brotherhood!.md)).

- [Cicero's Gloves](Skyrim_Cicero%27s_Gloves.md)
- [Jester's Gloves](Skyrim_Jester%27s_Gloves.md)
