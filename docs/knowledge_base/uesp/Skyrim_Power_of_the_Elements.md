Power of the Elements
---
*There are five versions of this book, all with the exact same name in-game, but containing different numbers of pages. The displayed information is for the most complete version of the book, `MGRDestruction04`. The other versions are:*

- `MGRDestruction01` (`0009C8C0`) contains only the first page (through "the sea will boil")
- `MGRDestruction02` (`0009C8C6`) contains only the first two pages (through "gild this stand with frost")
- `MGRDestruction03` (`0009C8C1`) contains only the first three pages (through "shake these snowcaps down")
- `MGRDestruction Final` (`000F37D0`) acts as a Spell Tome for [Fire Storm](Skyrim_Fire_Storm.md), and gets replaced with `MGRDestruction04` when read
