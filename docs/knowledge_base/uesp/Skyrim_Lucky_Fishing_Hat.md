## Related Quests
- **[Caught in the Rain](Skyrim_Caught_in_the_Rain.md)**<sup>[CC](Skyrim_Fishing.md)</sup>: Catch rainy weather fish.

## Notes
- If you return the hat during [Caught in the Rain](Skyrim_Caught_in_the_Rain.md) instead of lying, you can not get it back as the NPC is essential; there is no way to pickpocket the item back.
- The hat has the same enchantment as the [Ring of Kynareth](Skyrim_Ring_of_Kynareth.md).

## Bugs
- If you return the hat while still wearing it, the rain might not stop pouring even if you get another one via the console. The shout Clear Skies seems to stop the rain.
