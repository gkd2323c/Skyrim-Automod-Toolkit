# Weather

Different types of weather occur in [Skyrim](Skyrim_Skyrim.md), which can range from light rain to heavy winds of snow. In most cases, the weather will have no effect on the player character.

## Rain
Rain is a common weather event in Skyrim; it mostly occurs in forested areas such as [Falkreath](Skyrim_Falkreath.md), and it has a light tapping sound. Using the dragon shout [Clear Skies](Skyrim_Clear_Skies.md) will stop nearby rain in the area when it is used. When the [Fishing](Skyrim_Fishing.md) creation is installed, some varieties of fish can only be [caught](Skyrim_Fishing_(activity).md) when it is raining.

## Snow
Snow is everywhere in the northern regions of Skyrim, such as [Winterhold](Skyrim_Winterhold.md) or [Dawnstar](Skyrim_Dawnstar.md). It is usually very heavy with big winds accompanying it. Using the dragon shout, [Clear Skies](Skyrim_Clear_Skies.md) will stop nearby snowing in the area when it is used.

## Fog
[![](https://images.uesp.net/thumb/d/d2/SR-place-Hjaalmarch.jpg/200px-SR-place-Hjaalmarch.jpg)](https://en.uesp.net/wiki/File:SR-place-Hjaalmarch.jpg) [](https://en.uesp.net/wiki/File:SR-place-Hjaalmarch.jpg) Foggy saltmarsh of Hjaalmarch Fog is found in [Morthal](Skyrim_Morthal.md), it is used for a gloomy aesthetic and clouds the player's vision. Using the dragon shout [Clear Skies](Skyrim_Clear_Skies.md) can clear the surrounding area of fog.

When it appears, it will have a variety of color palettes.
