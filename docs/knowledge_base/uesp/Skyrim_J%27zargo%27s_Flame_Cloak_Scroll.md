# Scrolls

**Scrolls** are magical documents that can be used to cast spells. Anyone can use a scroll, regardless of [magic](Skyrim_Magic_Overview.md) skill, and casting a scroll does not use any [magicka](Skyrim_Magicka.md). However, each scroll can only be used once. With the exception of the Scroll of Blizzard, using a scroll does not provide any magic [skill](Skyrim_Skills.md) experience.

You cannot learn the spell contained in a scroll to cast later, you need a [spell tome](Skyrim_Spell_Tomes.md) for that.

Some scrolls can be purchased from spell [merchants](Skyrim_Merchants.md), who will have a random selection of standard scrolls they might sell. Note that scrolls are often found hiding in between rolls of paper, so it might be worthwhile scanning through them. Most scrolls are not found in random treasure, but are only available as fixed loot.

Scrolls can only be sold to spell merchants and general good merchants.

## Using Scrolls
To use a scroll, first select it in the inventory (or via the Favorites quick menu). This assigns it to a hand. Some scrolls require both hands. The assigned hand glows with a color or animation related to the scroll's spell.

Then, hold down the [attack button](Skyrim_Controls.md). A spell-casting animation takes place, during which the attack button must continue to be held down until it finishes. For some scrolls, this may take several seconds. Higher-level scrolls take longer to cast. Unlike with casting learned spells, the player cannot walk or run during the casting. Upon release of the attack button, the scroll's spell is unleashed. If the caster is interrupted (e.g. by a weapon blow or by a shout) before the casting is complete, then the spell is not cast, the scroll is not consumed, and the caster must start casting from the beginning. This can make it difficult to use a scroll in close combat.

## [Spider Scrolls](Skyrim_Spider_Scrolls.md)<sup>[DB](Skyrim_Dragonborn.md)</sup>
Dragonborn adds a special type of scrolls called spider scrolls, which can only be created in the [imbuing chamber](Skyrim_White_Ridge_Barrow.md) in [White Ridge Barrow](Skyrim_White_Ridge_Barrow.md). These can be equipped in either hand and are thrown rather than cast. As with standard scrolls, anyone can use a spider scroll regardless of [magic](Skyrim_Magic_Overview.md) skill, and using one does not use any [magicka](Skyrim_Magicka.md); however, each spider scroll can only be used once.

## Notes
- Destruction scrolls benefit from the damage-boosting perks in the Destruction skill tree. This is most notable with the Fire Storm scroll.
- [Followers](Skyrim_Followers.md) will also use scrolls if they are placed in their inventories, though this is limited to the conjure fire/frost/storm atronach scrolls and the Destruction scrolls.
- Various [leveled lists](Skyrim_Leveled_Lists.md) indicate that scrolls were originally going to be more prolific in Skyrim, particularly scrolls of the lower-tier spells. Most of these scrolls are [unobtainable](Skyrim_Unobtainable_Items.md#Scrolls).
- If you equip a scroll, transform into a [werewolf](Skyrim_Lycanthropy.md) or [Vampire Lord](Skyrim_Vampire_Lord.md)<sup>[DG](Skyrim_Dawnguard.md)</sup>, then drop the scroll after the transformation expires, the scroll will remain equipped and you can cast it as many times as you like.

## Bugs
- The Vendor Item Scroll keyword is missing from the following scrolls, making it impossible for spell merchants to buy or sell them: Bane of the Undead, Banish Daedra, Blizzard, Call to Arms, Calm, Candlelight, Chain Lightning, Circle of Protection, Command Daedra, Conjure Familiar, Conjure Flame Atronach, Conjure Frost Atronach, Conjure Storm Atronach, Courage, Dead Thrall, Dragonhide, Dread Zombie, Ebonyflesh, Expel Daedra, Fast Healing, Guardian Circle, Harmony, and Mass Paralysis. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.3.2, fixes this bug.
- Scrolls that reanimate a corpse will allow for the body to remain intact after the spell wears off, allowing the corpse to be reanimated an unlimited number of times. This is unlike reanimation spells, where the body will disintegrate after the spell wears off. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.9, addresses this issue. Reanimated corpses will disintegrate after the spell wears off.
- The condition checks for being in combat for the Flame Cloak Scroll, Frost Cloak Scroll, and Lightning Cloak Scroll are missing. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.1.3, fixes this bug.
- The keyword checks to make sure the Turn Undead Scroll, Turn Lesser Undead Scroll, Repel Lesser Undead Scroll, Repel Undead Scroll, and Turn Greater Undead Scroll only work on undead are missing. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.1.3, fixes this bug.
- The Fury Scroll is missing the secondary courage effect. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.1.3, fixes this bug.
- Call To Arms and Rally scrolls don't match the described effects they both claimed to apply. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Legendary Edition Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Legendary_Edition_Patch), version 3.0.4, fixes this bug.
- [Urag gro-Shub](Skyrim_Urag_gro-Shub.md) has no scrolls in his vendor inventory despite clearly stating he sells them when asked about it. - The [Unofficial Skyrim Special Edition Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Special_Edition_Patch), version 4.2.4, fixes this bug.
