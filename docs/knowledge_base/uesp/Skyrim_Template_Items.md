## Dummy Items
Dummy items are used as placeholders for random items of some type. Each occurrence in the game is associated with a [leveled list](https://en.uesp.net/wiki/Skyrim:Leveled_Lists) defining the possible items they get replaced with. The dummy items themselves have no name defined, and are therefore invisible.
