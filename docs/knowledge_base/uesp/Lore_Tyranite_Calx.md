# Tyranite Calx

[![](https://images.uesp.net/thumb/a/a7/ON-furnishing-Dormant_Tyranite_Calx.jpg/200px-ON-furnishing-Dormant_Tyranite_Calx.jpg)](https://en.uesp.net/wiki/File:ON-furnishing-Dormant_Tyranite_Calx.jpg) [](https://en.uesp.net/wiki/File:ON-furnishing-Dormant_Tyranite_Calx.jpg) Dormant Tyranite Calx Tyranite Calx is an interstitial alloy used in the production of [Dark Anchors](Lore_Dark_Anchors.md), infused with [soul gem fragments](Lore_Soul_Gems.md) and other trace components. Sondivel experimented with the metal using a [soul magic](Lore_Soul_Magic.md) ritual to shape it into malleable bands that could weaken the will of the wearer.<sup>[[1]](#cite_note-RJ-1)</sup><sup>[[2]](#cite_note-SJ-2)</sup>
