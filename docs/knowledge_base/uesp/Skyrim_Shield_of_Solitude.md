[![](https://images.uesp.net/thumb/b/be/SR-item-Shield_of_Solitude.jpg/200px-SR-item-Shield_of_Solitude.jpg)](https://en.uesp.net/wiki/File:SR-item-Shield_of_Solitude.jpg) [](https://en.uesp.net/wiki/File:SR-item-Shield_of_Solitude.jpg) Shield of Solitude The Shield of Solitude is a shield given to you by [Falk Firebeard](Skyrim_Falk_Firebeard.md) for saving Skyrim from the return of the Wolf Queen in [The Wolf Queen Awakened](Skyrim_The_Wolf_Queen_Awakened.md). It is identical in appearance to a [Solitude Guard's Shield](Skyrim_Solitude_Guard%27s_Shield.md). Its enchantment grants a [Resist Magic](Skyrim_Resist_Magic.md) and [Fortify Blocking](Skyrim_Fortify_Blocking.md) effect, which reduces the damage taken from offensive spells and reduces the damage taken if you successfully [block](Skyrim_Block.md) an attack.

## Related Quests
- **[The Wolf Queen Awakened](Skyrim_The_Wolf_Queen_Awakened.md)**: Potema must be stopped.

## Bugs
- This item can be disenchanted for a more powerful version of "Resist Magic" which tops at 25% (with enchant skill at 100) and can be stacked with the base Resist Magic enchantment of 20% using the Extra Effect to create a single item with 45% Resist Magic. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.0, fixes this bug.
- This item cannot be tempered. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, addresses this issue. Tempering the shield now requires an [iron ingot](Skyrim_Iron_Ingot.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.
