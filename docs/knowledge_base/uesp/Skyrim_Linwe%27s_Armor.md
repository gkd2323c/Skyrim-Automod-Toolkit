[![](https://images.uesp.net/thumb/6/65/SR-item-Linwe%27s_Armor.jpg/200px-SR-item-Linwe%27s_Armor.jpg)](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Armor.jpg) [](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Armor.jpg) Linwe's Armor Linwe's Armor is the chest piece of the unique set of equipment used by the leader of the [Summerset Shadows](Skyrim_Summerset_Shadows.md), [Linwe](Skyrim_Linwe.md). Alongside this item, this set of equipment also includes a pair of [boots](Skyrim_Linwe%27s_Boots.md), [gloves](Skyrim_Linwe%27s_Gloves.md), and a [hood](Skyrim_Linwe%27s_Hood.md). As Linwe runs an organization that rivals the [Thieves Guild](Skyrim_Thieves_Guild_(faction).md), his equipment has some visual similarities to the versions used by the Thieves Guild. Compared to [Thieves Guild Armor](Skyrim_Thieves_Guild_Armor.md), Linwe's Armor is grayer in color, but also sleeveless. Additionally, while it weighs one more unit than the Thieves Guild Armor, it also provides an additional two points of armor. However, its gold value is significantly lower. Besides that, the basic designs are the same. The armor is enchanted to [fortify stamina](Skyrim_Fortify_Stamina.md) by fifteen points. [Tempering](Skyrim_Workbench.md) Linwe's Armor requires a piece of [leather](Skyrim_Leather.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.

## Bugs
- While using Linwe's Armor there is a visual rendition glitch while an arrow is nocked or casting a spell and both your arms are invisible behind the gauntlet. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.2, fixes this bug.
- Linwe's Armor, Boots, Hood, and Gloves can all be disenchanted, and their respective enchantments named Shadowthrive, Shadowstrength, Shadowsight, and Shadowstrike learned, but these enchantments cannot be applied to any items during enchanting, making the resulting XP gain from disenchanting the only benefit for doing so. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, fixes this bug.
- Installing [Official Patch 1.9](Skyrim_Patch.md) can cause Linwe's Armor and the rest of the set to no longer benefit from the Matching Set bonus. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.2, addresses this issue. Bethesda overlooked his armor when working on the Official Patch. The error has now been reversed and the Matching Set bonus has been reinstated.
