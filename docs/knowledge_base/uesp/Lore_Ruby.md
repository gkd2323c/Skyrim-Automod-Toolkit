# Ruby

[![](https://images.uesp.net/thumb/e/e6/SR-item-Ruby_Geode.jpg/200px-SR-item-Ruby_Geode.jpg)](https://en.uesp.net/wiki/File:SR-item-Ruby_Geode.jpg) [](https://en.uesp.net/wiki/File:SR-item-Ruby_Geode.jpg) Ruby geode Ruby is a kind of red [crystal](Lore_Crystals.md). High-ranking members of the [Gray Host](Lore_Gray_Host.md) are known to wear earrings featuring rubies soaked in [vampire](Lore_Vampire.md) ichor.<sup>[[1]](#cite_note-1)</sup>
