# Magic Items with Guaranteed Locations

| --- |

There are a small number of **guaranteed generic magic items** that can be found in [Skyrim](Skyrim_Skyrim.md). Most generic magic items ([Generic Magic Apparel](Skyrim_Generic_Magic_Apparel.md) and [Generic Magic Weapons](Skyrim_Generic_Magic_Weapons.md)) are only found in randomized loot. The items listed on this page, however, are not randomly determined and will always be placed in the specified locations at the start of the game.

In most cases, detailed information about how to find each item is located on its corresponding location page.
