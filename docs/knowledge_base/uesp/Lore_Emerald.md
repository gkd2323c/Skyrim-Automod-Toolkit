# Emerald

[![](https://images.uesp.net/thumb/8/87/SR-item-Emerald_Geode.jpg/200px-SR-item-Emerald_Geode.jpg)](https://en.uesp.net/wiki/File:SR-item-Emerald_Geode.jpg) [](https://en.uesp.net/wiki/File:SR-item-Emerald_Geode.jpg) Emerald geode Emerald is a kind of green [crystal](Lore_Crystals.md). It is extremely rare and valuable. It also possesses modest, innate magical properties.<sup>[[1]](#cite_note-1)</sup><sup>[[2]](#cite_note-2)</sup>
