# Treasure Maps

[![](https://images.uesp.net/thumb/a/a9/SR-book-Treasure_Map_IV.png/200px-SR-book-Treasure_Map_IV.png)](https://en.uesp.net/wiki/File:SR-book-Treasure_Map_IV.png) [](https://en.uesp.net/wiki/File:SR-book-Treasure_Map_IV.png) Treasure Map IV The following table lists the various **map notes** found within the game. Most maps provide hints of a location where a chest with valuable items can be found. The chest will not be available until you have picked up the corresponding treasure map.
