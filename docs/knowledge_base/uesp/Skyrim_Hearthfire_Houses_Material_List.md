This page contains a *master list* of all the **materials** needed in [constructing](Skyrim_Construction.md), and fully furnishing, any of the three homes added in the [Hearthfire](Skyrim_Hearthfire.md) [add-on](Skyrim_Add-ons.md): [Heljarchen Hall](Skyrim_Heljarchen_Hall.md), [Lakeview Manor](Skyrim_Lakeview_Manor.md), and [Windstad Manor](Skyrim_Windstad_Manor.md).

The list will contain the differences between the materials required depending on which combination of the nine wings you want to add to your homestead. If you, instead, require a more detailed breakdown of each section per homestead, you might want to go to [this page](Skyrim_Construction.md).

## Notes
- The unique features found in the [exterior](Skyrim_Construction.md#Exterior) of each homestead would make this table too complicated, so they have been omitted. The materials needed for each are: - [Heljarchen Hall](Skyrim_Heljarchen_Hall.md) - 2 quarried stone, 2 sawn logs, and 2 iron fittings
- [Lakeview Manor](Skyrim_Lakeview_Manor.md) - 2 sawn logs, 1 nail, and 2 straw
- [Windstad Manor](Skyrim_Windstad_Manor.md) - 3 quarried stone, 3 sawn logs, and 4 nails
- Likewise, this list does not include the shrines located in the cellar as they all have unique materials. Instead, see [Shrines](Skyrim_Main_Hall.md#Shrines) for more information regarding the required materials.
