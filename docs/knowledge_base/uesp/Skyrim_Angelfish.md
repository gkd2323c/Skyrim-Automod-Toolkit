[![](https://images.uesp.net/thumb/f/fa/SR-ingredient-Angelfish.png/200px-SR-ingredient-Angelfish.png)](https://en.uesp.net/wiki/File:SR-ingredient-Angelfish.png) [](https://en.uesp.net/wiki/File:SR-ingredient-Angelfish.png) Angelfish **Angelfish** is an ingredient added by the [Fishing](Skyrim_Fishing.md) [Creation](Skyrim_Creation_Club.md).

Angelfish are rare and can only be caught when [fishing](Skyrim_Fishing_(activity).md) with an [Alik'ri Fishing Rod](Skyrim_Alik%27ri_Fishing_Rod.md) in temperate lakes. Two guaranteed Angelfish can be found in a pool within [Ancestor Glade](Skyrim_Ancestor_Glade.md) and can be caught without a fishing rod.
