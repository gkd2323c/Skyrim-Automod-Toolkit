## Bugs
- Because Glover's Letter respawns, you can obtain unlimited Exquisite Sapphires by repeatedly giving Sapphire the letter. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Legendary Edition Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Legendary_Edition_Patch) fixes this bug.
