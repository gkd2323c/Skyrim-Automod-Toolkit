# Atronach Forge Recipe

An **Atronach Forge Recipe** is any written information that explains the ingredients and any additional equipment or requirements needed to create things using the Atronach Forge. There are 3 recipes found in the [Atronach Forge Manual](https://en.uesp.net/wiki/Lore:Atronach_Forge_Manual), 9 in the [Report on Atronach Forge Experiments](https://en.uesp.net/wiki/Skyrim:Report_on_Atronach_Forge_Experiments), 2 that become available upon installation of additional content, and 31 different recipes that are found on paper notes throughout the original game. For a more thorough table of all 45 recipes, see the page: [Atronach Forge.](Skyrim_Atronach_Forge.md)
