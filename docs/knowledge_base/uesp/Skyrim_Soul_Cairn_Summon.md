**Soul Cairn Summon** is a shout unique to [Durnehviir](Skyrim_Durnehviir_(dragon).md), and therefore cannot be learned by the player aside from using [console](Skyrim_Console.md) commands. It summons one [Boneman](Skyrim_Boneman.md), one [Mistman](Skyrim_Mistman.md), and one [Wrathman](Skyrim_Wrathman.md) with each consecutive word, summoning one of each with all three words. When Durnehviir is summoned he will often use this shout while in combat, making these the only times the shout is used in-game.

Since the number of summoned creatures is normally limited to one, the shout requires the [Multiple Summon](https://en.uesp.net/wiki/Skyrim:Multiple_Summon) perk to function correctly.

## Notes
- Despite being voiced by Durnehviir, if the player adds this shout through console commands the shout will not be voiced.
