# Bend'r-mahk

The Bend'r-mahk (sometimes capitalized as Bend'r-Mahk) is a word with an unknown definition and designation.<sup>[[1]](#cite_note-DC-1)</sup><sup>:9</sup><sup>[[2]](#cite_note-SKManual-2)</sup><sup>[[3]](#cite_note-OBLathon-3)</sup><sup>[[nb 1]](#intnote_nb1)</sup> Though it is strongly associated in some way with the [Western Reach](Lore_Western_Reach.md)<sup>[[2]](#cite_note-SKManual-2)</sup><sup>[[4]](#cite_note-PGE3High-4)</sup> as well as the [Redguard](Lore_Redguard.md) [Crowns](Lore_Crowns.md).<sup>[[5]](#cite_note-PGE3H-5)</sup><sup>[[UOL 1]](#cite_note-Vivecmeeting-6)</sup>
