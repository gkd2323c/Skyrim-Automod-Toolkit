# Ellenica

| Saint Ellenica | | | |
| --- | --- | --- | --- |
| Race | [Breton](Lore_Breton.md) | Gender | Female |

**Saint** Ellenica was a priestess of Arkay.<sup>[[1]](#cite_note-ESOAlvaren-1)</sup> Her prayer book is a holy relic, and contains passages that can be read to put the dead to rest.<sup>[[2]](#cite_note-2)</sup> She was buried in Flyleaf Catacombs with her prayer book, which was retrieved in [2E 582](Lore_Second_Era.md#2E_582).<sup>[[3]](#cite_note-3)</sup>
