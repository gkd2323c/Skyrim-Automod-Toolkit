[†](#note_Studded) Although described in-game as Imperial "medium" armor, it is actually light armor. - The studded Imperial armor cannot be tempered at a workbench. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, fixes this bug.

[††](#note_penitusoculatushelmet) This item is one of only three items in the game (along with [Falmer Helmets](Skyrim_Falmer_Helmet.md) and the [Jagged Crown](Skyrim_Jagged_Crown.md)) that fills only the "hair" body part slot when worn. This means that it can be worn simultaneously with any [circlet](Skyrim_Circlet.md) or a [Dragon Priest Mask](Skyrim_Dragon_Priest_Mask.md). Out of the three items, the Penitus Oculatus Helmet is the most lightweight. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.2, fixes this bug.

[‡](#note_IH) The open-faced Imperial Helmet does not use the Steel Smithing perk. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.2, fixes this bug.
