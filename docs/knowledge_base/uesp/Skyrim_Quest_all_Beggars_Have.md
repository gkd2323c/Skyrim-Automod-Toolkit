## Walkthrough
If you talk to one of the quest givers, they will ask you for some gold. Giving them one gold piece completes the quest and rewards you with the blessing [The Gift of Charity](Skyrim_The_Gift_of_Charity.md).

## Notes
- This quest does not have any stages or objectives.
- This quest can be repeated indefinitely.
- You will not have the option to give a beggar gold for four in-game hours after you last gave a beggar gold, or if you have no gold.
- Because this is a favor quest which affects the quest giver's disposition, it will count toward the "Help the People" part of the [thane](Skyrim_Thane.md) quest for the quest giver's hold.
