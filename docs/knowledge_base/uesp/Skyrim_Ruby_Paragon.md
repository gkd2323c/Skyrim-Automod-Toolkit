[![](https://images.uesp.net/thumb/8/83/SR-item-Ruby_Paragon.jpg/200px-SR-item-Ruby_Paragon.jpg)](https://en.uesp.net/wiki/File:SR-item-Ruby_Paragon.jpg) [](https://en.uesp.net/wiki/File:SR-item-Ruby_Paragon.jpg) The Ruby Paragon The Ruby Paragon is located on a [frost giant](Skyrim_Frost_Giant.md) in the [Inner Sanctum](Skyrim_Inner_Sanctum_(Vale).md) of the Forgotten Vale, behind a door that can only be opened with the [Initiate's Ewer](Skyrim_Initiate%27s_Ewer.md). If placed into the gem holder of the [Paragon Platform](Skyrim_Forgotten_Vale.md#Paragon_Platform), it will create a portal to [Forgotten Vale Forest](Skyrim_Forgotten_Vale_Forest.md), an area with Frost Trolls fighting a [Falmer](Skyrim_Falmer.md) that carries an [artifact](Skyrim_Artifacts.md), [Auriel's Shield](Skyrim_Auriel%27s_Shield.md).
