- This location is one of many potential targets for [one or more of the Radiant quests](Skyrim_Radiant.md#Quest_Locations).

## Walkthrough
A set of wooden ramps leads up to the door to the two-story watchtower. An Orc bandit guards the entrance, with another Orc bandit inside on the ground floor. The ground floor features a pile of firewood and a [tanning rack](Skyrim_Tanning_Rack.md), a table holding a mead barrel, a [leg of goat](https://en.uesp.net/wiki/Skyrim:Leg_of_Goat), two [leather strips](https://en.uesp.net/wiki/Skyrim:Leather_Strips), two [garlic](Skyrim_Garlic.md) bulbs, a [rock warbler egg](https://en.uesp.net/wiki/Skyrim:Rock_Warbler_Egg), two [raw rabbit legs](https://en.uesp.net/wiki/Skyrim:Raw_Rabbit_Leg), and a copy of *[The Code of Malacath](https://en.uesp.net/wiki/Skyrim:The_Code_of_Malacath)*, and an open fire and a cooking spit with a charred [skeever](Skyrim_Skeever.md) cooking over it which you can harvest for a [charred skeever hide](https://en.uesp.net/wiki/Skyrim:Charred_Skeever_Hide).

A stone spiral staircase leads to the top level, which is guarded by an Orc bandit and the Orc bandit chief. The top level features an Orc-styled shelter with two chairs and a long wooden table holding a joint of [raw beef](https://en.uesp.net/wiki/Skyrim:Raw_Beef), [grilled leeks](https://en.uesp.net/wiki/Skyrim:Grilled_Leeks), a bottle of [Nord mead](https://en.uesp.net/wiki/Skyrim:Nord_Mead), an enchanted [Orcish warhammer](https://en.uesp.net/wiki/Skyrim:Orcish_Warhammer), a copy of the [Heavy Armor](Skyrim_Heavy_Armor.md) [skill book](Skyrim_Skill_Books.md) *[Hallgerd's Tale](https://en.uesp.net/wiki/Skyrim:Hallgerd%27s_Tale)*, and a copy of *[The True Nature of Orcs](https://en.uesp.net/wiki/Skyrim:The_True_Nature_of_Orcs)*. Behind the table is an unlocked chest, and to the left is a small table holding a random [bow](Skyrim_Bow.md) and [arrows](https://en.uesp.net/wiki/Skyrim:Arrows).

## Notes
- There is an [orichalcum ore vein](https://en.uesp.net/wiki/Skyrim:Orichalcum_Ore_Vein) to the southeast.

## Bugs
- The Orc bandit chief does not respawn, which makes it impossible to complete [radiant quests](Skyrim_Radiant.md) at this location if he has been killed previously. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim:Mod_Unofficial_Skyrim_Patch), version 2.0.1, fixes this bug.

- [![On PC](https://images.uesp.net/d/d7/Computer.svg)](https://www.wikipedia.org/wiki/PC_game) Open the [console](Skyrim_Console.md), then use the command `prid d1f56` followed by `resurrect`. This should revive the Orc and allow you to complete the quest as normal.
