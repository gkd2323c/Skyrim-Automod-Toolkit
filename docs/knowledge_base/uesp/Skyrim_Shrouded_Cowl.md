[![](https://images.uesp.net/thumb/8/8f/SR-item-Shrouded_Cowl.jpg/200px-SR-item-Shrouded_Cowl.jpg)](https://en.uesp.net/wiki/File:SR-item-Shrouded_Cowl.jpg) [](https://en.uesp.net/wiki/File:SR-item-Shrouded_Cowl.jpg) Shrouded Cowl A Shrouded Cowl serves as the helmet for a set of armor used by the [Dark Brotherhood](Skyrim_Dark_Brotherhood.md). Alongside this veil, this set of armor also includes [armor](Skyrim_Shrouded_Armor.md), [boots](Skyrim_Shrouded_Boots.md), and [gloves](Skyrim_Shrouded_Gloves.md). This cowl [fortifies archery](Skyrim_Fortify_Archery.md) by twenty points, increasing damage done with bows. [Tempering](Skyrim_Workbench.md) a Shrouded Cowl requires a piece of [leather](Skyrim_Leather.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.

## Bugs
- The maskless version of this armor cannot be improved at a workbench. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, fixes this bug.
- The male version of the normal Shrouded Cowl does not have a mask.
