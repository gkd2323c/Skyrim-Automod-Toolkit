[![](https://images.uesp.net/thumb/3/3c/SR-item-Linwe%27s_Boots.jpg/200px-SR-item-Linwe%27s_Boots.jpg)](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Boots.jpg) [](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Boots.jpg) Linwe's Boots Linwe's Boots serve as the foot protection for the unique set of equipment used by the leader of the [Summerset Shadows](Skyrim_Summerset_Shadows.md), [Linwe](Skyrim_Linwe.md). Alongside this item, this set of equipment also includes [armor](Skyrim_Linwe%27s_Armor.md), a pair of [gloves](Skyrim_Linwe%27s_Gloves.md), and a [hood](Skyrim_Linwe%27s_Hood.md). As Linwe runs an organization that rivals the [Thieves Guild](Skyrim_Thieves_Guild_(faction).md), his equipment has some visual similarities to the versions used by the Thieves Guild. Compared to [Thieves Guild Boots](Skyrim_Thieves_Guild_Boots.md), Linwe's Boots are grayer in color. Additionally, while his boots weigh half a unit more than the Thieves Guild Boots, it also provides an additional two points of armor, as well as being worth several hundred more gold pieces. Besides that, the basic designs of the items are the same. The armor is enchanted to [fortify sneaking](Skyrim_Fortify_Sneak.md) by fifteen points. [Tempering](Skyrim_Workbench.md) Linwe's Boots requires a piece of [leather](Skyrim_Leather.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.

## Bugs
- Linwe's Armor, Boots, Hood, and Gloves can all be disenchanted, and their respective enchantments named Shadowthrive, Shadowstrength, Shadowsight, and Shadowstrike learned, but these enchantments cannot be applied to any items during enchanting, making the resulting XP gain from disenchanting the only benefit for doing so. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, fixes this bug.
- Installing [Official Patch 1.9](Skyrim_Patch.md) can cause Linwe's Boots and the rest of the set to no longer benefit from the Matching Set bonus. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.2, addresses this issue. Bethesda overlooked his armor when working on the Official Patch. The error has now been reversed and the Matching Set bonus has been reinstated.
