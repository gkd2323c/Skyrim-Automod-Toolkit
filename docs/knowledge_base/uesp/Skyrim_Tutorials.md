# Tutorials


Note that you can use crafting skills without doing any of the tutorial quests, and the quests themselves do not give you any advantage in using the skills.

- **[Alchemy Tutorial](https://en.uesp.net/wiki/Skyrim:Alchemy_Tutorial)**: Learn the basics of potion making. ([radiant](Skyrim_Radiant.md))
- **[Blacksmithing Tutorial](https://en.uesp.net/wiki/Skyrim:Blacksmithing_Tutorial)**: Learn the basics of [smithing](Skyrim_Smithing.md). ([radiant](Skyrim_Radiant.md))
- **[Enchanting Tutorial](https://en.uesp.net/wiki/Skyrim:Enchanting_Tutorial)**: Use an [Arcane Enchanter](Skyrim_Arcane_Enchanter.md).
