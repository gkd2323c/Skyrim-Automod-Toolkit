## Bugs
- The two Labyrinthian load screens will display randomly for the remainder of the game once triggered, rather than only during the relevant quests, as seems to have been intended. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim:Mod_Unofficial_Skyrim_Patch), version 2.0.1, fixes this bug.
- The New Game and Blindsighted load screens will do the same and also display randomly for the remainder of the game once triggered, instead of only during the relevant quests or times. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim:Mod_Unofficial_Skyrim_Patch), version 2.0.5, fixes this bug.
- The two load screens added by The Cause are set to display once the global variable cc BGSSSE067_Show Load Screens is set to 1, but nowhere is it changed from its default value of 0. <sup>**?**</sup>
- You might get loading screens even when you don't fulfill the criteria for seeing them (such as Thieves Guild screens when you're not even part of the Thieves Guild). <sup>**?**</sup>
