# The Improved Emperor's Guide to Tamriel

The Improved Emperor's Guide to Tamriel **by [Flaccus Terentius](Lore_Flaccus_Terentius.md)** *The first draft of an illustrated guidebook depicting the regions of Tamriel, circa 2E 578*
---
**The Improved Emperor's Guide to Tamriel** was a 224-page illustrated guide included with physical copies of the [Imperial Edition](https://en.uesp.net/wiki/Online:Imperial_Edition) of *[The Elder Scrolls Online](https://en.uesp.net/wiki/Online:Online)*. The timeframe of the guide is contemporary to the game, which itself is set in 2E 582.

**You can read about the Guide [here](https://en.uesp.net/wiki/Books:The_Improved_Emperor%27s_Guide_to_Tamriel).**
