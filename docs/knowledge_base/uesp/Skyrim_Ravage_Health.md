*Alchemy description:* **Causes <mag> points of concentrated poison damage.**
This effect is a variant of [Damage Health](Skyrim_Damage_Health.md). It causes a flat total reduction of <mag> points of target's **maximum** [Health](Skyrim_Health.md) points instantaneously.

As with the other two Damage Health variants, it is only available from poisons.

## Notes
- The effects of Ravage Health can be reversed by altars and [Cure Poison](Skyrim_Potions.md#Cure_Poison) potions.
- Additionally, Ravage Health poisons are a good choice against opponents with Restoration magic, since there is no Cure Poison spell, they usually don't carry Cure Poison potions, and there is no spell to simply heal the lost maximum health.
- Ravage Health is extremely potent on higher difficulties, because it is not technically damage, and therefore is not affected by the damage multiplier. On Legendary difficulty, it is four times more effective than it appears to be.
- Because Ravage Health has a relatively short duration, it is best used for finishing a fight. If used at the start of a fight, it will typically wear off before the target is dead. However, it *can* stack multiple times if applied quickly enough.

## Bugs
- The potion has a duration, but the description only says "Causes x points of concentrated poison damage". - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.0, addresses this issue. The potion's description now specifies how long the duration is.
- The description "Concentrated poison drains Health by x points per second for x seconds" doesn't match the potion's effect. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.9, addresses this issue. The description now says "Concentrated poison temporarily reduces maximum and current Health by x points for x seconds."
