# Nix-Hound Items

This page documents the **new items** added by the [Nix-Hound](Skyrim_Nix-Hound.md) [Creation](Skyrim_Creation_Club.md).

## Food

### Raw Food
The Creation takes care to add the raw Nix-Hound Meat to [Survival Mode](Skyrim_Survival_Mode.md)'s raw food table, so you can contract [Food Poisoning](Skyrim_Disease.md#Food_Poisoning) by consuming the uncooked variant.
