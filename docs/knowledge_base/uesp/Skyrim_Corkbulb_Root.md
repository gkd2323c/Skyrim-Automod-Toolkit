**Corkbulb Root** is an ingredient added by the [Rare Curios](Skyrim_Rare_Curios.md) [Creation](Skyrim_Creation_Club.md). It can be purchased from [Khajiit caravans](Skyrim_Khajiit_Traders.md).

This ingredient can also be used to fashion [corkbulb arrows](Skyrim_Corkbulb_Arrow.md) and [bolts](Skyrim_Corkbulb_Bolt.md).
