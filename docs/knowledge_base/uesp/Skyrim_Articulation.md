**Articulation** is functionally equivalent to [Fortify Barter](Skyrim_Fortify_Barter.md).

## Bugs
- Articulation technically counts as a Fortify Sneak effect, rather than a Fortify Barter effect, though there are no cases in the game or official add-ons where this makes a difference. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.1, fixes this bug.
