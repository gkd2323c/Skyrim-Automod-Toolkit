[![](https://images.uesp.net/thumb/1/1a/SR-item-Linwe%27s_Gloves.jpg/200px-SR-item-Linwe%27s_Gloves.jpg)](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Gloves.jpg) [](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Gloves.jpg) Linwe's Gloves Linwe's Gloves serve as the gauntlets for the unique set of equipment used by the leader of the [Summerset Shadows](Skyrim_Summerset_Shadows.md), [Linwe](Skyrim_Linwe.md). Alongside this item, this set of equipment also includes [armor](Skyrim_Linwe%27s_Armor.md), a pair of [boots](Skyrim_Linwe%27s_Boots.md), and a [hood](Skyrim_Linwe%27s_Hood.md). As Linwe runs an organization that rivals the [Thieves Guild](Skyrim_Thieves_Guild_(faction).md), his equipment has some visual similarities to the versions used by the Thieves Guild. Compared to [Thieves Guild Gloves](Skyrim_Thieves_Guild_Gloves.md), Linwe's Gloves are grayer in color. Additionally, while his gloves weigh one unit more than the Thieves Guild Gloves, it also provides an additional two points of armor, as well as being worth several hundred more gold pieces. Besides that, the basic designs of the items are the same. The armor is enchanted to [fortify one-handed](Skyrim_Fortify_One-Handed.md) attacks by fifteen points. [Tempering](Skyrim_Workbench.md) Linwe's Gloves requires a piece of [leather](Skyrim_Leather.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.

## Bugs
- Linwe's Armor, Boots, Hood, and Gloves can all be disenchanted, and their respective enchantments named Shadowthrive, Shadowstrength, Shadowsight, and Shadowstrike learned, but these enchantments cannot be applied to any items during enchanting, making the resulting XP gain from disenchanting the only benefit for doing so. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, addresses this issue. Linwe's items can't be disenchanted.
- Installing [Official Patch 1.9](Skyrim_Patch.md) can cause Linwe's Gloves and the rest of the set to no longer benefit from the Matching Set bonus. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.2, addresses this issue. Bethesda overlooked his armor when working on the Official Patch. The error has now been reversed and the Matching Set bonus has been reinstated.
