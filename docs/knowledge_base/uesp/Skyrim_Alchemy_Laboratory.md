# Alchemy Laboratory


The **Alchemy Laboratory** is one of the North Wing additions you can [build](Skyrim_Construction.md) to expand your home from the [Main Hall](https://en.uesp.net/wiki/Skyrim:Main_Hall). The tower can hold a large variety of storage options, as well as many alchemy-dedicated bins and containers. Plenty of [ingredients](Skyrim_Ingredients.md) are spread about when the tower is fully furnished.

The Alchemy Laboratory, like its counterparts, the [Library](Skyrim_Library.md) and [Enchanter's Tower](Skyrim_Enchanter%27s_Tower.md), has a balcony area on its roof when completed. A roof canopy can be added to these towers if desired, but not adding the final roof canopy does not prevent the tower from being inhabited, allowing it to be an optional addition. Once you have built the roof, then you can build all interior items in the tower.

The following is a list of possible furniture items and decorations which can be added by using the carpenter's workbench in the Alchemy Laboratory . It is not necessary to purchase or build all furnishing options; you can pick and choose to better dictate style. However, all furnishing options are in a pre-arranged location and only ingredients and standard items can be moved. Oftentimes, the name of the furnishing option only indicates the primary piece of furniture, and actually includes various additional [containers](Skyrim_Containers.md) and accessories.

The "Level" column in the table notes the level on which the furnishing option is placed: **1** for the downstairs level, **2** for the upstairs level, and **R** for the rooftop balcony. ## Notes
- The Alchemy Laboratory is not required to have an [alchemy table](Skyrim_Alchemy_Labs.md) in the house, as one can be built in the Main Hall as well.
- If you leave the canopy off of the tower balcony, you'll be provided with a full, unobstructed view of the surrounding landscape. The tower will still function like normal, and anyone living on the property will still use it as part of their daily routines.

## Bugs
- Building the alchemy laboratory requires housebuilding materials, i.e. locks, hinges, and iron fittings, for three doors, even though the tower only has two doors connecting to the main hall, unlike the towers that can be built for the east and west wings.
- Sometimes you may build an item, have it placed, and have it reappear as an item you can build again in the worktable. When selecting this item the second time, your resources are used, but no item is made. However, there is no need to select it a second time since it has not disappeared. Simply re-enter the worktable and the erroneous selection will not be offered.
- Sometimes it will show that the Alchemy table is built, when it hasn't. Simply remove and then replace the worktable and it should work. <sup>**?**</sup>
- The pair of wall sconces on the wall supports on level 1 will not have the sconce built on the northeast support at Lakeview Manor. <sup>**?**</sup>
- The Alchemy table in the tower can only be used when activating the circular rim of the table. You cannot interact with the rest of the table (the alembic, the green spot in the center, etc.) to open the potion creation menu because the option to do so doesn't appear.
