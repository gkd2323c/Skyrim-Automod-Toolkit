## Quick Walkthrough
1. Gain entrance to the [College of Winterhold](Skyrim_College_of_Winterhold_(place).md).
2. Read the [research notes](Skyrim_Telekinesis_Arrows_Research_Notes.md) and take the [spell tome](Skyrim_Spell_Tome__Telekinesis_Arrows.md) from [the Arcanaeum](Skyrim_The_Arcanaeum.md).

## Detailed Walkthrough
**Note: This quest was originally added to your journal upon installing the [Arcane Archer Pack](Skyrim_Arcane_Archer_Pack.md) [Creation](Skyrim_Creation_Club.md). With the release of the [Anniversary Edition](Skyrim_Anniversary_Edition.md), this quest no longer appears in your journal. The [Spell Tome](Skyrim_Spell_Tome__Telekinesis_Arrows.md) and [Note](Skyrim_Telekinesis_Arrows_Research_Notes.md) added by the Creation can still be acquired from [the Arcanaeum](Skyrim_The_Arcanaeum.md) in the [College of Winterhold](Skyrim_College_of_Winterhold_(place).md), and [Telekinesis Arrows](Skyrim_Telekinesis_Arrow.md) can still be crafted.**

Once you've installed the Arcane Archer Pack Creation, you will receive this quest directing you to the College of Winterhold to learn more about a spell tome which has been uncovered. The tome in question, [*Spell Tome: Telekinesis Arrows*](Skyrim_Spell_Tome__Telekinesis_Arrows.md), is held in the Arcanaeum, requiring you to first gain access to the college in order to proceed. See the [First Lessons](Skyrim_First_Lessons.md) quest walkthrough for more information on how to join the college.

Locate the spell tome on the counter in front of [Urag gro-Shub](Skyrim_Urag_gro-Shub.md). Simply picking it up will advance the quest. However, you must read the tome in order to learn the related [Telekinesis Arrows](Skyrim_Telekinesis_Arrows.md) lesser power, which must be used to control the arrows. Reading the nearby [*Telekinesis Arrows Research Notes*](Skyrim_Telekinesis_Arrows_Research_Notes.md) will teach you how to craft [Telekinesis Arrows](Skyrim_Telekinesis_Arrow.md), provides an introductory description of how the arrows function and completes the quest.
