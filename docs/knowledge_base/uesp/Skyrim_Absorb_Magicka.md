*Enchanting description:* **Absorb <mag> points of magicka.**

**Absorb Magicka** (a.k.a. **Drain Magicka**) transfers <mag> points of [Magicka](Skyrim_Magicka.md) from the target to the person wielding the weapon. On the target, the effect is comparable to [Damage Magicka](Skyrim_Damage_Magicka.md); on the attacker, the effect is comparable to [Restore Magicka](https://en.uesp.net/wiki/Skyrim:Restore_Magicka). [Spell Absorption](Skyrim_Spell_Absorption.md) is similar to Absorb Magicka, in that your magicka can be increased at the expense of your enemy, but is a passive effect rather than an active effect.

Two versions of Absorb Magicka are actually available for enchanting. The stats above are for the most common version of the effect. A second version is only available from one item: the [Drainspell Bow](Skyrim_Drainspell_Bow.md). The bow can be disenchanted, meaning that its effect can be used for custom enchantments. The stats for this second version of the effect are:

- ID: 0010f221
- Cost: 1.2

## Enchanting
The following items use this effect. You can learn how to enchant custom items with Absorb Magicka if you find one of the following items and disenchant it:

- All weapons *[of the Sorcerer](Skyrim_Generic_Magic_Weapons.md#Absorb_Magicka)*
- All weapons *of Evoking*
- All weapons *of Harrowing*
- All weapons *of Siphoning*
- All weapons *of Winnowing*
- [Gauldur Blackbow](Skyrim_Gauldur_Blackbow.md)
- [Drainspell Bow](Skyrim_Drainspell_Bow.md) - has unique version of effect

Artifacts and unique items that use the effect but cannot be disenchanted include:

- [Cleaver of St. Felms](Skyrim_Cleaver_of_St._Felms.md)<sup>[CC](Skyrim_Ghosts_of_the_Tribunal.md)</sup>
- [Dwarven Black Bow of Fate](Skyrim_Dwarven_Black_Bow_of_Fate.md)<sup>[DB](Skyrim_Dragonborn.md)</sup>
- [Harkon's Sword](Skyrim_Harkon%27s_Sword.md)<sup>[DG](Skyrim_Dawnguard.md)</sup> when inflicted with [Vampirism](Skyrim_Vampirism.md)
- [Keening](Skyrim_Keening.md)
- [Ruin's Edge](Skyrim_Ruin%27s_Edge_(item).md)<sup>[CC](Skyrim_Ruin%27s_Edge.md)</sup>
- [Soulrender](https://en.uesp.net/wiki/Skyrim:Soulrender)<sup>[DB](Skyrim_Dragonborn.md)</sup> when wielded with [Bloodscythe](Skyrim_Bloodscythe.md)<sup>[DB](Skyrim_Dragonborn.md)</sup>

## Notes
- In Skyrim, there appears to be no distinction between "Drain Magicka" and "Absorb Magicka".
- Absorb Magicka will not work against [Dwarven automatons](Skyrim_Dwarven_Automatons.md).

## Bugs
- The name of the enchanting effect appears to be mistakenly set to "Drain Magicka" instead of "Absorb Magicka", even though the description and editor ID both identify it as "Absorb". - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.3.2, fixes this bug.
