The **Unofficial Skyrim Special Edition Patch** (**USSEP**) is a major patch that fixes numerous bugs left over even after the [latest official patch](Skyrim_Patch.md) is applied to [Skyrim](Skyrim_Skyrim.md). These fixes range in importance from dialogue typos to quest-breaking bugs to reducing save file bloat, and cover both the vanilla game and the official [Dawnguard](Skyrim_Dawnguard.md), [Hearthfire](Skyrim_Hearthfire.md), and [Dragonborn](Skyrim_Dragonborn.md) add-ons.

The USSEP is incompatible with the Legendary Edition of the game; the older **Unofficial Skyrim Legendary Edition Patch** (**USLEEP** or **USLEP**) patch should be used for that edition. Likewise the USLEEP patch is incompatible with the Special Edition of the game. See [where to download](#Where_to_Download) for links.

The Unofficial Patch was originally packaged in a series of discrete patches specific to the aforementioned game components and designed to work in concert: **Unofficial Skyrim Patch**, **Unofficial Dawnguard Patch**, and **Unofficial Dragonborn Patch**. After the release of the Legendary Edition of the game which included all three add-ons, these separate patches were discontinued in favor of the unified USLEEP patch which required all three of the aforementioned add-ons and was first formally released on November 7, 2015. Unification made upkeep significantly easier and enabled several fixes to be implemented that could not be done with discrete patches. Though the standalone patches are now essentially obsolete (and have been removed from most download sites), they can be used in a case of running just the original base game, or the base game plus one DLC but not the other.

An **Unofficial High Resolution Patch** exists, which fixes bugs for the official [High Resolution add-on](Skyrim_High_Resolution_Texture_Pack.md). and is designed to work alongside USLEEP.

With the release of Skyrim Special Edition, the patches diverged into the USLEEP and the USSEP, with the first release under the USSEP acronym on October 28, 2016. The USSEP incorporated the **Unofficial High Resolution Patch** making that patch defunct for the Special Edition.

The major fixes are being gradually added to UESP. Should you want to help point out such a fix, please use the {{[Bug](https://en.uesp.net/wiki/Template:Bug)}} template to do so. Specifically, you should use {{[Bug](https://en.uesp.net/wiki/Template:Bug) | *Details of bug.* |USKP| *version*}}. Optionally, you can also add a fourth parameter to give more information on any notable details about the fix. For other variations or special cases, see the template's documentation.

Note that this patch is *unofficial*. The *official* (Bethesda-released) patches are documented at [Patch](Skyrim_Patch.md) and [Special Edition Patch](Skyrim_Special_Edition_Patch.md); the official patches include some fixes that for technical reasons (e.g., altering the game's executable) cannot be incorporated into the unofficial patch patches. Therefore, installing the official patches is still required. Also, the unofficial patch patches are not available for Xbox 360, PS3, and Switch players, as they cannot install any third-party content.
