[![](https://images.uesp.net/thumb/1/10/SR-npc-Umbra.jpg/200px-SR-npc-Umbra.jpg)](https://en.uesp.net/wiki/File:SR-npc-Umbra.jpg) [](https://en.uesp.net/wiki/File:SR-npc-Umbra.jpg) Umbra Umbra is the main antagonist of the [Umbra](Skyrim_Umbra.md) creation. Previously an Imperial named **Cresitus**, he is now the host for the [Umbra](Skyrim_Umbra_(item).md) Daedric artifact, which has taken over his identity. He wears a set of Ebony armor, wields Umbra and carries a [Champion's Rest Passage Key](Skyrim_Champion%27s_Rest_Passage_Key.md).
