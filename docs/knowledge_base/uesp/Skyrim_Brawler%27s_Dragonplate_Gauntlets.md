# Brawler's Dragonplate Gauntlets

**Brawler's Dragonplate Gauntlets** are gauntlets added by the [Fearsome Fists](Skyrim_Fearsome_Fists.md) [Creation](Skyrim_Creation_Club.md). There are two variations of this item — the unenchanted version can be crafted at a forge and differs from regular [Dragonplate Gauntlets](Skyrim_Dragonplate_Armor.md) only in appearance, whereas the enchanted version has an added [Fortify Unarmed Damage](Skyrim_Fortify_Unarmed_Damage.md) effect, and cannot be crafted but can be bought from vendors and found as random loot from level 40.<sup>[*verification needed — level*]</sup>
