Namira's Itch is a [unique](Skyrim_Unique_Items.md) [war axe](Skyrim_War_Axe.md) added by [Alternative Armors - Dragon Plate](Skyrim_Alternative_Armors_-_Dragon_Plate.md). It can be obtained from [Alvasorr the Rat](Skyrim_Alvasorr_the_Rat.md). The axe has the same appearance as the [Falmer war axe](Skyrim_Falmer_War_Axe.md), but has a higher base [damage](Skyrim_Damage.md). Tempering Namira's Itch is not possible.

## Related Quests
- **[Bones for a Crow](Skyrim_Bones_for_a_Crow.md)**<sup>[CC](Skyrim_Alternative_Armors_-_Dragon_Plate.md)</sup>: Hunt down the Forsworn Crowstooth.

## Notes
- It can be disenchanted.

## Bugs
- It uses the drawing sound effect of a sword.
- Similar to [Keening](Skyrim_Keening.md), its enchantment has a charge capacity of zero, making it unusable under normal circumstances.
