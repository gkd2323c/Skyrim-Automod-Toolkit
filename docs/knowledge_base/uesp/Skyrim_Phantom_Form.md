**Phantom Form** is a [dragon shout](Skyrim_Dragon_Shouts.md) that creates a [phantom](https://en.uesp.net/wiki/Skyrim:Phantom). It is used by the [Greybeards](Skyrim_Greybeards.md) to create targets for shout practice.

## Related Quests
- **[The Way of the Voice](Skyrim_The_Way_of_the_Voice.md)**: Meet the [Greybeards](Skyrim_Greybeards.md).

## Notes
- It is unavailable to the player, but PC users can use [console](Skyrim_Console.md) commands to learn and unlock the shout. This shout was apparently intended to be usable by the player, as it is voiced for all races and its three Word Walls can be found in the game files.
- Although the unused Word Wall writes "SAH" as S4, using a single glyph for ah, the third word of the shout itself actually uses separated a and h letters, rendering it as SAH.
