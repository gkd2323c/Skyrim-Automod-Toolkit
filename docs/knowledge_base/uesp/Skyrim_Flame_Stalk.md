[![](https://images.uesp.net/thumb/f/fe/SR-flora-Flame_Stalk.jpg/200px-SR-flora-Flame_Stalk.jpg)](https://en.uesp.net/wiki/File:SR-flora-Flame_Stalk.jpg) [](https://en.uesp.net/wiki/File:SR-flora-Flame_Stalk.jpg) Flame Stalks Flame Stalk is an ingredient added by the [Rare Curios](Skyrim_Rare_Curios.md) [Creation](Skyrim_Creation_Club.md) and the [Saints & Seducers](Skyrim_Saints_%26_Seducers.md) Creation. It can be purchased from [Khajiit caravans](Skyrim_Khajiit_Traders.md). If the [Saints & Seducers](Skyrim_Saints_%26_Seducers.md) Creation is installed, it can also be harvested from Flame Stalk plants, which are only found in [Solitude Sewers](Skyrim_Solitude_Sewers.md). Flame Stalks obtained from both sources don't stack in the inventory, and it's possible to combine both in a single potion<sup>[*verification needed — The Saints & Seducers version of the Flame Stalk ingredient should only be obtainable if Rare Curios is not installed*]</sup>.
