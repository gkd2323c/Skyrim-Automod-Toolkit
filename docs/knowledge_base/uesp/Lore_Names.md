This section lists all known **names** for the various [races](Lore_Races.md) (and certain [creatures](Lore_Bestiary.md)) featured in the *[Elder Scrolls](https://en.uesp.net/wiki/General:The_Elder_Scrolls)* series. Due to the number of names for playable races, as well as for Maormer, Reachmen, and Daedra, each of these has its own page. All of the other races have their names listed here.

Names are organized by type, source, and frequency. The type can be male names, female names, or sometimes family names and titles. The source may be a game, out-of-game book, or lore for names mentioned only in in-game books. Some races have names with observed patterns that subdivide them further. For example, [Argonian](Lore_Argonian.md) names are 1-word, hyphenated, or Cyrodilic.
