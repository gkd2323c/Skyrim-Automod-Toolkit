**Powerful summoned and raised creatures are put under your control.**

## Enchanting
Only one item is enchanted with this effect and it cannot be disenchanted:

- [Staff of Daedric Command](https://en.uesp.net/wiki/Skyrim:Staff_of_Daedric_Command)

## Spells
Only one spell uses this effect, and the effect has no variants:

- [Command Daedra](https://en.uesp.net/wiki/Skyrim:Command_Daedra): *Command* (`0006f954`; `Command FFActor`)

## Notes
- This is one of the few instances in the game where the same underlying effect is used for both a spell and an enchantment.
