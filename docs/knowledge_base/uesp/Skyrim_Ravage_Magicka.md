*Alchemy description:* **Concentrated poison damages maximum magicka by <mag> points.**
This effect is a variant of [Damage Magicka](Skyrim_Damage_Magicka.md). It causes a total of <mag> points of damage, instantaneously.

As with the other Damage Magicka variant, [Lingering Damage Magicka](Skyrim_Lingering_Damage_Magicka.md), it is only available from poisons.

## Notes
- Ravage Magicka poisons are a good choice against mages since there is no spell to simply heal the lost maximum Magicka.
- Because Ravage Magicka has a relatively short duration, it is best used for finishing a fight with a mage. If used at the start of a fight, it will typically wear off before the target is dead. However, it *can* stack multiple times if applied quickly enough.

- The potion has a duration, but the description only says "Concentrated poison damages maximum Magicka by x points."- ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.0, addresses this issue. The potion's description now specifies how long the duration is.
- The description "Concentrated poison drains Magicka by x points per second for x seconds" doesn't match the potion's effect. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.9, addresses this issue. The description now says "Concentrated poison temporarily reduces maximum and current Magicka by x points for x seconds."
