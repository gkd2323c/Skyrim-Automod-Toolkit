*Enchanting description:* **<mag> points of extra damage to Animals.**

## Enchanting
Only one item uses this effect. It can be disenchanted, allowing you to create custom items with the Huntsman's Prowess effect:

- [Poacher's Axe](https://en.uesp.net/wiki/Skyrim:Poacher%27s_Axe)

## See Also
This effect is related to:

- [Animal Damage](https://en.uesp.net/wiki/Skyrim:Animal_Damage)

## Notes
- While the Poacher's Axe has no charge meter (and therefore never needs recharging), any items enchanted with the Huntsman's Prowess effect learned from disenchanting the Poacher's Axe will have a charge meter and will require recharging.
