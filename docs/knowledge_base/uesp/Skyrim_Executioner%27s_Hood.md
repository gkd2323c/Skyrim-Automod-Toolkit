[![](https://images.uesp.net/thumb/5/5b/SR-item-Executioner%27s_Hood.jpg/200px-SR-item-Executioner%27s_Hood.jpg)](https://en.uesp.net/wiki/File:SR-item-Executioner%27s_Hood.jpg) [](https://en.uesp.net/wiki/File:SR-item-Executioner%27s_Hood.jpg) Executioner's Hood An Executioner's Hood is a shroud worn by [headsmen](Skyrim_Headsman.md). The Executioner's Hood is part of a set of items that also includes a set of [boots](Skyrim_Unobtainable_Items.md#000cf8b0), [gloves](Skyrim_Unobtainable_Items.md#000cf8b1), and [robes](Skyrim_Executioner%27s_Robes.md#000cf8b3). Besides providing a grim appearance for the wearer, it also provides some anonymity to one plying such a grisly trade. The Executioner's Hood has been enchanted to [fortify magicka](Skyrim_Fortify_Magicka.md) by forty points, allowing the wearer to cast more spells. However, the Executioner's Hood cannot be equipped, and no copy can be obtained during normal gameplay.
