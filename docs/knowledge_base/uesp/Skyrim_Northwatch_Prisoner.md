# Northwatch Prisoner

The **Northwatch Prisoners** can be found in the northeastern part of [Northwatch Keep](Skyrim_Northwatch_Keep.md). All three are locked inside separate cells and merely function as supposed cell mates when [Thorald Gray-Mane](Skyrim_Thorald_Gray-Mane.md) spawns in the interrogation chamber as part of the [related quest](#Related_Quests).

All three prisoners wear simple clothing, a [roughspun tunic](Skyrim_Roughspun_Tunic.md) with a pair of [footwraps](Skyrim_Footwraps.md).

## Bugs
- None of the prisoners will attempt to escape from captivity, even if the dungeon is cleared and the cell doors are unlocked. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.2, addresses this issue. The prisoners now flee once rescued.
