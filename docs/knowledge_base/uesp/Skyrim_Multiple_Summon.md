# Multiple Summon

**Multiple Summon** is a perk active on the undead dragon [Durnehviir](Skyrim_Durnehviir_(dragon).md)<sup>[DG](Skyrim_Dawnguard.md)</sup> in Dawnguard add-on, with which he is able to summon a [boneman](Skyrim_Boneman.md), a [mistman](Skyrim_Mistman.md), and a [wrathman](Skyrim_Wrathman.md) all at once by using his unique [Soul Cairn Summon](Skyrim_Soul_Cairn_Summon.md) shout.

| Perk | Editor ID | [Form ID](Skyrim_Form_ID.md) |
| --- | --- | --- |
| Multiple Summon | DLC1Durnehviir Multiple Summon | xx01a33c |

Without this perk, even if you obtain the Soul Cairn Summon shout with [console](Skyrim_Console.md) commands, and using all the three words, you can only summon one undead. This perk changes the amount of creatures you can have to 5, similar to how the [Twin Souls](Skyrim_Twin_Souls.md) perk changes the limit to 2. Thus, it applies to all conjuration and summon powers, shouts and spells. With this perk, you can even have 5 NPCs revived with Dead Thrall.

This perk can only be acquired by the player through the console.
