*Alchemy description:* **Carrying capacity increases by <mag> for <dur> seconds.**
*Enchanting description:* **Carrying capacity increased by <mag> points.**
**Fortify Carry Weight** increases the target's maximum [Carry Weight](Skyrim_Carry_Weight.md), thereby increasing the total weight that can be carried. It is similar to the Feather effect in previous Elder Scrolls games.

[Potions](Skyrim_Potions.md) (custom and built-in) provide a temporary increase that only lasts for a specified duration, after which the target's carry weight is returned to normal, possibly causing the target to be over-encumbered. [Enchantments](Skyrim_Enchanting_Effects.md) last for as long as the item is equipped, but are generally smaller in magnitude than potions.

The following items use this effect. You can learn how to [enchant](Skyrim_Enchanting.md) customs with this effect if you find one of the following items and disenchant it.

- [Charmed Necklace](https://en.uesp.net/wiki/Skyrim:Charmed_Necklace)
- All varieties of Boots *[of Lifting](Skyrim_Generic_Magic_Apparel.md#Fortify_Carry_Weight)*, *of Hauling*, *of Strength*, *of Brawn*, *of the Ox*, and *of the Mammoth*
