# Magic Script

This alphabet is seen in the Master Illusion Texts in the [College of Winterhold](Lore_College_of_Winterhold.md), on Shalidor's Insights and the Oghma Infinium. It has connections to the [Mages Guild](Lore_Mages_Guild.md), and some of the symbols in it resemble others used by the guild. There are only five known texts using this alphabet, and as of yet there is no known translation for any of them. However, as the symbols are given in font-form, rather than images, we can see which letter each character represents by using the [Creation Kit](https://en.uesp.net/wiki/Skyrim_Mod:Creation_Kit). The name "Magic Script" is only seen in the Creation Kit, as no name for the alphabet is given within the games.

Some of the symbols also resemble others used by the Mages guild.

- The letter 'K' ([![MAGICSCRIPT K.png](https://images.uesp.net/e/ed/MAGICSCRIPT_K.png)](https://en.uesp.net/wiki/File:MAGICSCRIPT_K.png)) resembles the symbol used to represent the school of Destruction, as seen on skill books and spell tomes.
- The letter 'E' ([![MAGICSCRIPT E.png](https://images.uesp.net/9/91/MAGICSCRIPT_E.png)](https://en.uesp.net/wiki/File:MAGICSCRIPT_E.png)) resembles the symbol for the Mysticism skill seen in Oblivion.

## Shalidor's Insights
[![](https://images.uesp.net/thumb/e/e1/SR-book-Arcane00.png/200px-SR-book-Arcane00.png)](https://en.uesp.net/wiki/File:SR-book-Arcane00.png) [](https://en.uesp.net/wiki/File:SR-book-Arcane00.png) A page from Shalidor's Insights The book [Shalidor's Insights](Skyrim_Shalidor%27s_Insights_(book).md) actually has some translatable words using the Magic Script. The block reads:

```
ALDUIN A
VA RI A O
NODICA DE
AKATOSH E
S SUPERFI
C I A L
M E N T E
PAREC ES E
COM O SEU
COLEGA NOS

```
The passage is actually Portuguese: "Alduin é a variação Nórdica de Akatosh e só superficialmente parece-se com o seu colega nos", meaning "Alduin is the Nordic variation of Akatosh, and superficially resembles his counterpart in the [Nine Divines]", the same as the first somewhat-legible English paragraph on the same page and a sentence from *[Varieties of faith in the Empire](Lore_Varieties_of_Faith....md)*.

On another page (which features are stylized golden spiral, as seen here: [Arcane Scribblings p.3](http://host.conseiljedi.com/~wolf/Skyrim/arcane%20scribblings/arcane04.png)), the columns down the left- and right-hand sides contain some translatable words. First, the left-hand side can be transcribed top to bottom as:

```
QACONDUZIUOAQUI AESTEP

```
This string of letters contains the word **conduziu**, which is Portuguese for *(it has) led* (the 3rd person singular perfect past tense of **conduzir** *to lead*), **o aqui** *(to) here* and **a este** *to this*. Putting this together: *...led to here, to this...". The right-hand column is transcribed thus: *

```
QESKYRIMSEPTIMUSVIVEEMU

```
In this string, we find *Skyrim* and *Septimus* (presumably referencing Septimus Signus) and then **vive em** which is again Portuguese, for *live(s) in...*. Potentially, this could be rendered as: *Septimus lives in Skyrim* (ignoring the three seemingly obsolete letters QE and U and also ignoring normal Portuguese word order, which is normally subject-verb-object).

The letters around the spiral are (from the outside in): STUVW. This seems to be meaningless, simply being in consecutive alphabetical order. The text beneath is illegible and in a different script.

Another page featuring a symbol similar to that of destruction magic (as seen [here](http://i39.tinypic.com/szvl9u.png)) also contains a passage using Magic Script. It is a transliteration of English:

```
ALDUIN S CIVILI
ZATION WAS THE
DRAGON CULT OF
ATMORA HE S BA
SICALLY THE
DRAGON GOD ON
EARTH THE
DRAGON HIGH
PRIESTS ARE HIS
ACOLYTES AND
EVERYTHING IS
ORDERED

```
Rendered in plain text as: *Alduin's civilization was the Dragon Cult of Atmora. He's basically the Dragon God on Earth. The Dragon High Priests are his acolytes and everything is ordered.* The large glyph in the top left is an "H".
