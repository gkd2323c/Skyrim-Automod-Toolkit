[![](https://images.uesp.net/thumb/1/1c/SR-item-Linwe%27s_Hood.jpg/200px-SR-item-Linwe%27s_Hood.jpg)](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Hood.jpg) [](https://en.uesp.net/wiki/File:SR-item-Linwe%27s_Hood.jpg) Linwe's Hood Linwe's Hood is the headwear for the unique set of equipment used by the leader of the [Summerset Shadows](Skyrim_Summerset_Shadows.md), [Linwe](Skyrim_Linwe.md). Alongside this item, this set of equipment also includes [armor](Skyrim_Linwe%27s_Armor.md), a pair of [boots](Skyrim_Linwe%27s_Boots.md), and a [gloves](Skyrim_Linwe%27s_Gloves.md). As Linwe runs an organization that rivals the [Thieves Guild](Skyrim_Thieves_Guild_(faction).md), his equipment has some visual similarities to the versions used by the Thieves Guild. However, his hood is the exception to the rest of the equipment. While a [Thieves Guild Hood](Skyrim_Thieves_Guild_Hood.md) is a brown and rounded at the corners, this hood has much sharper appearance, is more patchwork than the Thieves Guild version, and is also gray. Additionally, while his hood weighs half a unit more than the Thieves Guild Hood, it also provides an additional three points of armor, as well as being slightly more valuable. The armor is enchanted to [fortify Archery](Skyrim_Fortify_Archery.md) attacks by fifteen points. [Tempering](Skyrim_Workbench.md) Linwe's Hood requires a piece of [leather](Skyrim_Leather.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.

## Bugs
- Linwe's Armor, Boots, Hood, and Gloves can all be disenchanted, and their respective enchantments named Shadowthrive, Shadowstrength, Shadowsight, and Shadowstrike learned, but these enchantments cannot be applied to any items during enchanting, making the resulting XP gain from disenchanting the only benefit for doing so. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, fixes this bug.
- Installing [Official Patch 1.9](Skyrim_Patch.md) can cause Linwe's Hood and the rest of the set to no longer benefit from the Matching Set bonus. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.2, addresses this issue. Bethesda overlooked his armor when working on the Official Patch. The error has now been reversed and the Matching Set bonus has been reinstated.
