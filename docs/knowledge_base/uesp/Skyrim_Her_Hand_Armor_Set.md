# Her Hand Armor Set

Her Hand Armor Set is an equipment set worn by Almalexia's personal guardians. Although Almalexia is gone, the armor is still used by members of a cult dedicated to the Tribunal. As it was intended for the personal guards of a god, the set consists entirely of [heavy armor](Skyrim_Heavy_Armor.md). Every piece of the set requires a [chitin plate](Skyrim_Chitin_Plate.md) to be improved.

There is both an enchanted and unenchanted version of this armor set. The enchanted version provides the following benefits:

- [Fortify Heavy Armor](Skyrim_Fortify_Heavy_Armor.md), 25 pts
- [Fortify Stamina](Skyrim_Fortify_Stamina.md), 70 pts
- [Fortify One-handed](Skyrim_Fortify_One-Handed.md), 40 pts
- [Fortify Block](Skyrim_Fortify_Block.md), 40 pts
