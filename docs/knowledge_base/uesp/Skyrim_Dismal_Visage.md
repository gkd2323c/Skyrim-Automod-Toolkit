[![](https://images.uesp.net/thumb/a/af/SR-item-Dismal_Visage.jpg/200px-SR-item-Dismal_Visage.jpg)](https://en.uesp.net/wiki/File:SR-item-Dismal_Visage.jpg) [](https://en.uesp.net/wiki/File:SR-item-Dismal_Visage.jpg) Dismal Visage The Dismal Visage is a [unique](Skyrim_Unique_Items.md) [heavy](Skyrim_Heavy_Armor.md) [armor](Skyrim_Armor.md) added by [Alternative Armors - Dragon Plate](Skyrim_Alternative_Armors_-_Dragon_Plate.md). It can be obtained from [Alvasorr the Rat](Skyrim_Alvasorr_the_Rat.md). The helmet has the same appearance as the [Falmer helmet](Skyrim_Falmer_Helmet.md), but has a higher [armor rating](Skyrim_Armor_Rating.md). Improving it is not possible.

## Related Quests
- **[Bones for a Crow](Skyrim_Bones_for_a_Crow.md)**<sup>[CC](Skyrim_Alternative_Armors_-_Dragon_Plate.md)</sup>: Hunt down the Forsworn Crowstooth.

## Notes
- The **Dismal Visage** can be disenchanted.
- It is possible to equip this helmet alongside any circlet or Dragon Priest mask.

## Bugs
- Like the Falmer helmet, this helmet is affected by the [light armor](Skyrim_Light_Armor.md) perks.
