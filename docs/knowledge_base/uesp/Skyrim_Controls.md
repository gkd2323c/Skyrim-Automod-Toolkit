## Notes
- [![On PC](https://images.uesp.net/d/d7/Computer.svg)](https://www.wikipedia.org/wiki/PC_game) [Patch](Skyrim_Patch.md) Version 1.2.12.0 does not allow you to assign controls to number pad keys 1-8, returning the error message *"That button is reserved".*
- [![On PC](https://images.uesp.net/d/d7/Computer.svg)](https://www.wikipedia.org/wiki/PC_game) Remapping a key will remap it for all uses, but the game interface will continue to show the default control. For example if "Left Strafe" (A) is remapped to (Q), then you must use Q for lockpicking although the interface will continue to direct you to the A key for lockpicking. The "Change POV" key is similarly bound to the in-menu "Favorites" toggle.
- [![On Xbox](https://images.uesp.net/3/33/Xbox.svg)](Skyrim_Xbox.md) When switching the attack buttons (RT and LT) with (RB and LB) you will be unable to block properly and can only use the power or shout currently equipped.
- [![On Play Station](https://images.uesp.net/6/64/Playstation.svg)](Skyrim_PlayStation.md) To adjust the distance of the camera in third person mode, right click and hold the right analog stick, then move the left analog stick vertically.
- In Switch you can't toggle any buttons to RSR, RSL, LSR or LSL.
