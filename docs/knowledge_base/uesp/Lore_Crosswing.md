# Crosswing

| Crosswings | |
| --- | --- |
| Type | [Beast](Lore_Bestiary.md) (Arthropod) |
| Range | [Valenwood](Lore_Valenwood.md) |

Crosswings are [insects](Lore_Insect.md) that inhabit [Valenwood](Lore_Valenwood.md). They are believed to have a fierce bitterness when they die afraid. The [Bosmer](Lore_Bosmer.md) use spells to attract them as part of the process for brewing [beer](Lore_Beer.md) from meat.<sup>[[1]](#cite_note-OTBODMB-1)</sup>

## Notes
- Crosswings originally appeared in the unofficial text *[The Dreadful Theft of the Sun's Dusk Ale](https://en.uesp.net/wiki/General:The_Dreadful_Theft_of_the_Sun%27s_Dusk_Ale)*.<sup>[[UOL 1]](#cite_note-UOLTD-2)</sup>
