# Flipback

| Flipbacks | |
| --- | --- |
| Type | [Beast](Lore_Bestiary.md) (Arthropod) |
| Range | [Valenwood](Lore_Valenwood.md) |

Flipbacks are [insect](Lore_Insect.md) creatures that inhabit [Valenwood](Lore_Valenwood.md). The [Bosmer](Lore_Bosmer.md) use spells to repel them as part of the process for brewing [beer](https://en.uesp.net/wiki/Lore:Beer) from meat.<sup>[[1]](#cite_note-OTBODMB-1)</sup> They have peculiar "false eyes" on their thorax and can glow in the dark.<sup>[[UOL 1]](#cite_note-DGFP-2)</sup>

## Notes
- Flipbacks originally appeared in the unofficial text *[The Dreadful Theft of the Sun's Dusk Ale](https://en.uesp.net/wiki/General:The_Dreadful_Theft_of_the_Sun%27s_Dusk_Ale)*.<sup>[[UOL 2]](#cite_note-UOLTD-3)</sup>
- According to [Douglas Goodall](https://en.uesp.net/wiki/General:Douglas_Goodall), flipbacks were based off [eastern eyed click beetles](https://www.wikipedia.org/wiki/Alaus_oculatus).<sup>[[UOL 1]](#cite_note-DGFP-2)</sup>
