# Clairvoyance (effect)

| [](https://en.uesp.net/wiki/File:SR-icon-spell-Magic_Hat.png) | Clairvoyance |
| --- | --- |
| School | [Illusion](Skyrim_Illusion.md) |
| Type | Other |

**Shows the path to the current goal.**

**Clairvoyance** draws a path, in the form of a magical blue line, leading to your current quest marker or objective.

## Spells
Only one spell uses this effect, and the effect has no variants.

- [Clairvoyance](Skyrim_Clairvoyance_(spell).md): *Clairvoyance* (`0002113f`; `Clairvoyance Effect`)

## Notes
- Clairvoyance can be used to navigate to a player-placed map marker.
- Both Clairvoyance and your compass's quest marker show you where to head. However, the quest marker only shows you the straight-line direction, whereas Clairvoyance shows you the route, taking into account all twists and turns.
- Clairvoyance shows the path to the nearest waypoint marker only—no preference is given for markers on final destinations (not leading to an area transition point). For this reason it is of no use in finding the nearest objective of many quests. With several quests active, Clairvoyance will direct you to a doorway or other transition point, then repeatedly direct you through that point from either side. As long as you have a quest objective on either side of the door, the door itself is the nearest quest marker, rendering Clairvoyance useless. - If there are multiple targets with no intervening waypoint, such as in [With Friends Like These...](Skyrim_With_Friends_Like_These....md), Clairvoyance will target the one that was added to the quest objectives last.
- Clairvoyance does not take into account doors you cannot open when showing you the shortest path.
- Clairvoyance can be used in some cases for finding hidden objective points. This most commonly works with search and collect style objectives that do not have an objective marker.
- On low settings for the PC, the Clairvoyance effect may not be rendered.
- The Clairvoyance spell only draws a blue line as far as you can see, in-game. So, if you use [Fortify Illusion](Skyrim_Fortify_Illusion.md) enchantments or large [Fortify Magicka](Skyrim_Fortify_Magicka.md) enchantments to be able to cast Clairvoyance indefinitely, or for a long period of time, you will eventually reach the end of the blue line, and have to re-cast the spell. For this reason, it is impossible - even in the Creation Kit - to create a permanent Clairvoyance ability or enchantment.

## Bugs
- Clairvoyance can occasionally fail to work, creating a trail that when followed will eventually loop around and return to a location that had already been led to by the spell. This creates an endless loop. Alternatively, it will show the blue path only under where you stand, and can also lead to completely unrelated areas. - One such example is when using it to navigate to Northwatch Keep while doing the quest [Missing In Action](Skyrim_Missing_In_Action.md), it will lead instead to the entrance of Steepfall Burrow.
- Clairvoyance may fail to work altogether, never showing any target at all. Sometimes, this is triggered by specific quests being selected (e.g., [Hunt and Gather](Skyrim_Hunt_and_Gather.md)). - Unselect all quests in your journal. Set a manual marker. At this point, Clairvoyance should start working again. From here, you can remove the manual marker and re-select quests as normal. Pay attention as you select quests to see if any of them trigger the problem again.
- Saving the game while Clairvoyance is active may make the shown path remain permanently.
