**Chokeberry** is an ingredient added by the [Rare Curios](Skyrim_Rare_Curios.md) [Creation](Skyrim_Creation_Club.md). It can be purchased from [Khajiit caravans](Skyrim_Khajiit_Traders.md), and [Babette](Skyrim_Babette.md) also sells 1-4 samples.

Chokeberries can also be used in [cooking](Skyrim_Cooking.md) to create [poisoned green apples](https://en.uesp.net/wiki/Skyrim:Poisoned_Green_Apple) and [poisoned red apples](https://en.uesp.net/wiki/Skyrim:Poisoned_Red_Apple), which can be powerful tools of assassination.
