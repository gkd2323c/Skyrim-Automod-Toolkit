Note to Rhorlak **by [Rigel Strong-Arm](Skyrim_Rigel_Strong-Arm.md)** *Concerning the theft of a silver mold from Markarth*

We got the right cart this time. The note has been sent and we'll see how they respond.

In the meantime, we'll keep breaking down this stuff - you keep selling off the [wood](Skyrim_Firewood.md) to keep our [front](Skyrim_Pinewatch.md). If we can get a good price for the [mold](Skyrim_Curious_Silver_Mold.md), we should be set for a while.

Oh and we left some [wine](Skyrim_Alto_Wine.md) for you in a bucket behind the shelf. Consider it a bonus.

-- [Rigel Strong-Arm](Skyrim_Rigel_Strong-Arm.md)
