[![](https://images.uesp.net/thumb/7/7a/SR-item-Iron_Battleaxe.jpg/200px-SR-item-Iron_Battleaxe.jpg)](https://en.uesp.net/wiki/File:SR-item-Iron_Battleaxe.jpg) [](https://en.uesp.net/wiki/File:SR-item-Iron_Battleaxe.jpg) The Woodsman's Friend [![](https://images.uesp.net/thumb/3/3c/SR-npc-Wood_Cutter_02.jpg/200px-SR-npc-Wood_Cutter_02.jpg)](https://en.uesp.net/wiki/File:SR-npc-Wood_Cutter_02.jpg) [](https://en.uesp.net/wiki/File:SR-npc-Wood_Cutter_02.jpg) The dead wood cutter the weapon is found near The Woodsman's Friend is a unique [iron battleaxe](Skyrim_Iron_Battleaxe.md) found alongside a dead [wood cutter](Skyrim_Wood_Cutter.md). With no enchantment and no way to [improve](Skyrim_Grindstone.md) the weapon through blacksmithing, the only advantage this weapon has over the generic version of this weapon is the single extra point of default damage it can do. However, as the weapon has a lower speed, the overall benefit of this extra point of damage is diminished. Additionally, it has approximately half of the gold value of a normal iron battleaxe, making it notably subpar stat-wise to the standard version of this item.

## Notes
- Despite the name, you cannot use the Woodsman's Friend to create [firewood](Skyrim_Firewood.md) at [wood chopping blocks](Skyrim_Chop_Wood.md).

## Bugs
- The Woodsman's Friend does no critical damage by default. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.2.6, addresses this issue. The weapon can now do eight points of critical damage.
- This battleaxe cannot be tempered, despite there not being an obvious reason for this. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.0, addresses this issue. It can now be improved with [iron ingots](Skyrim_Iron_Ingot.md).
- The battleaxe will respawn even though it is a unique weapon. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.1, fixes this bug.
- The Woodsman's Friend has its detection noise sound set to "normal" when it should be set to "loud" like a regular iron battleaxe. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Legendary Edition Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Legendary_Edition_Patch), version 3.0.8, fixes this bug.
