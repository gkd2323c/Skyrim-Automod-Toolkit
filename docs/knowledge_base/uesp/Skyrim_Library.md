# Library


The Library, like its counterparts, the [Alchemy Laboratory](Skyrim_Alchemy_Laboratory.md) and [Enchanter's Tower](Skyrim_Enchanter%27s_Tower.md), has a balcony area on its roof when completed. A roof canopy can be added to these towers if desired, but not adding the final roof canopy does not prevent the tower from being inhabited, allowing it to be an optional addition. Once you have built the roof, then you can build all interior items in the tower.

The following is a list of possible furniture items and decorations which can be added by using the carpenter's workbench in the Library. It is not necessary to purchase or build all furnishing options; you can pick and choose to better dictate style. However, all furnishing options are in a pre-arranged location and only ingredients and standard items can be moved. Oftentimes, the name of the furnishing option only indicates the primary piece of furniture, and actually includes various additional [containers](Skyrim_Containers.md) and accessories.

## Furnishing Options
The bookshelves in the library can hold up to 308 books total. A number of miscellaneous items are added with various furnishing options, but only notable items are specified on this table. This includes usable [containers](Skyrim_Containers.md), rare items, or items which were newly added by Hearthfire. Though they are added to your home, other common items or home accessories are not listed here.

The "Level" column in the table notes the level on which the furnishing option is placed: **1** for the downstairs level, **2** for the upstairs level, and **R** for the rooftop balcony.

## Notes
- If you build all 9 bookshelves, you can place a total of 308 books on them.
- If you leave the canopy off of the tower balcony, you'll be provided with a full, unobstructed view of the surrounding landscape. The tower will still function like normal, and anyone living on the property will still use it as part of their daily routines.
