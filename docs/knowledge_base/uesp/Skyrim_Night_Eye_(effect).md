**Improved night vision for <dur> seconds.**

**Night Eye** allows the caster to see in the dark, making the entire field of view brighter. It is of limited use during the day time.

The only way to acquire Night Eye in Skyrim is a [lesser power](https://en.uesp.net/wiki/Skyrim:Lesser_Power), meaning that only the following character types have access to it:

- [Khajiit](Skyrim_Khajiit.md), through the Night Eye [lesser power](https://en.uesp.net/wiki/Skyrim:Lesser_Power) (Form ID `000aa01d`)
- [Vampires](Skyrim_Vampirism.md), through the [Vampire's Sight](https://en.uesp.net/wiki/Skyrim:Vampire%27s_Sight) lesser power (Form ID `000c4de1`) - It is also a secondary effect of the [Embrace of Shadows](https://en.uesp.net/wiki/Skyrim:Embrace_of_Shadows) [greater power](https://en.uesp.net/wiki/Skyrim:Greater_Power) (Form ID `00088821`)
- [Werewolves](Skyrim_Werewolf.md), through their version of the Night Eye lesser power (Form ID `0006b10e`), though this power is not normally accessible.

As a lesser power, the effect can be cast as many times per day as desired.
