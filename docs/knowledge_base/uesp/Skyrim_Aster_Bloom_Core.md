**Aster Bloom Core** is an ingredient added by the [Rare Curios](Skyrim_Rare_Curios.md) [Creation](Skyrim_Creation_Club.md). It can be purchased from [Khajiit caravans](Skyrim_Khajiit_Traders.md). They can occasionally be found in [hollow stumps](https://en.uesp.net/wiki/Skyrim:Hollow_Stump) in the [Solitude Sewers](Skyrim_Solitude_Sewers.md) with the [Saints & Seducers](Skyrim_Saints_%26_Seducers.md) Creation installed.
