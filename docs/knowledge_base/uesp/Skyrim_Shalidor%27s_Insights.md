## Quick Walkthrough
1. Talk to [Urag gro-Shub](Skyrim_Urag_gro-Shub.md).
2. Travel to the indicated location and retrieve [Shalidor's Insights](Skyrim_Shalidor%27s_Insights_(book).md).
3. Return to Urag gro-Shub.

## Detailed Walkthrough
Talk to [Urag gro-Shub](Skyrim_Urag_gro-Shub.md) and ask him if there are any special books he's looking for. He will tell you that he's looking for a book Arch-Mage Shalidor wrote so he can translate his writings. The book can be located in the boss-level chest of any [dungeon](Skyrim_Dungeons.md) in the game including ones added in [Dawnguard](Skyrim_Dawnguard.md) and [Dragonborn](Skyrim_Dragonborn.md), meaning that your destination is randomly selected from over 140 possible dungeons. For specific details about the target location, see its place page; details about any possible conflicts with other quests are also provided there.

The book is in the dungeon's boss chest, normally located at the end of the dungeon. Return to [Urag gro-Shub](Skyrim_Urag_gro-Shub.md) and return the book. He will say that it will take about a day to translate the writings.

The quest will complete at this point and clear from your journal, although you will have written that you still need to return to Urag after the book has been translated.

Wait for at least 24 hours, and when you return Urag will tell you he's finished translating the text. He will give you three identical copies of one of several possible ["Shalidor's Insights" scrolls](Skyrim_Shalidor%27s_Insights__Alteration.md), or will pass on any knowledge gained from the writings in the form of a one-point skill increase to any magic skill other than Enchanting.

## Notes
- Which scrolls or skill boost you earn is decided when you hand over [Shalidor's Insights](Skyrim_Shalidor%27s_Insights_(book).md) to [Urag gro-Shub](Skyrim_Urag_gro-Shub.md). It is possible to obtain the reward you prefer by saving before giving him the book and reloading the save until the desired reward is obtained.
- If the quest marker points to [Druadach Redoubt](Skyrim_Druadach_Redoubt.md), it is possible to complete this quest very easily by doing the Forsworn chain in [Markarth](Skyrim_Markarth.md), as this is where Madanach has his base of operations. All of the Forsworn in this area are friendly if you completed [No One Escapes Cidhna Mine](Skyrim_No_One_Escapes_Cidhna_Mine.md), as long as you sided with the Forsworn in the mine.
- The quest may point to a number of locations which cannot be accessed without completing certain other actions. You may wish to avoid clashes by reloading the game and getting another location. Following is a list of locations and the actions needed to be undertaken to complete this quest: - [Rebel's Cairn](Skyrim_Rebel%27s_Cairn.md): partial completion of [Find Red Eagle's Sword](Skyrim_Find_Red_Eagle%27s_sword.md) / [Rebel's Cairn](Skyrim_Rebel%27s_Cairn_(quest).md).
- [Snow Veil Sanctum](Skyrim_Snow_Veil_Sanctum.md): complete [Speaking With Silence](Skyrim_Speaking_With_Silence.md). The quest marker may also point to the back door, but you still must complete Speaking with Silence.
- [The Katariah](Skyrim_The_Katariah.md): complete [Hail Sithis!](Skyrim_Hail_Sithis!.md) This location is the most common reason why the quest marker does not appear. (This is addressed by version 1.2 of the [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch); the Katariah can no longer be chosen as a target location for this quest.)
- [Ysgramor's Tomb](Skyrim_Ysgramor%27s_Tomb.md): complete [Glory of the Dead](Skyrim_Glory_of_the_Dead.md).
- If [Dragonborn](Skyrim_Dragonborn.md) is installed, some radiant locations may be on [Solstheim](Skyrim_Solstheim.md). If you are in Skyrim, you will not receive a quest marker pointing you to the quest location, but must travel to Solstheim first (at which point any quest markers you have to quests in Skyrim will disappear). - One of these locations can be [Nchardak](Skyrim_Nchardak.md), which means you would need to do [The Path of Knowledge](Skyrim_The_Path_of_Knowledge.md) to complete this side quest.

## Bugs
- If the quest marker points to [High Gate Ruins](Skyrim_High_Gate_Ruins.md) and you have previously defeated Vokun, the book will be in an inaccessible area in the back of Vokun's Throne Room. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.2.5, fixes this bug.

- Vokun's Throne Room may be accessed by utilizing a follower to open the gate blocking the player from the antechamber exit above the first room. This is accomplished by bringing a follower after clearing out the entire dungeon (clearing out the dungeon may be necessary), climbing up the pile the rubble, in the first room, besides Anska, and using Whirlwind Sprint to access the walkway leading to the exit from the antechamber. The follower will run through the dungeon and eventually appear on the other side of the gate. The follower can then be instructed to pull the chain, thereby opening the gate and permitting the player into Vokun's Throne Room. Unconfirmed on PC.
- [![On PC](https://images.uesp.net/d/d7/Computer.svg)](https://www.wikipedia.org/wiki/PC_game) This can be made reachable using [console](Skyrim_Console.md) commands, either to resurrect Vokun and kill him again (select the body, not his ashes) or to simply pass through one of the closed gates (use 'tcl' to turn clipping on and off).
- If the quest [Hitting the Books](Skyrim_Hitting_the_Books.md) is active when this quest sends you to [Fellglow Keep](Skyrim_Fellglow_Keep.md), the "insights" may be missing. - To fix this you can type in console `sqv mgr21`, look for a line REF 'Book'->"(x), x is id you need, then jump and open console again midair so the book won't fall under ground, type in console `prid x` where x is id you got previously, and last type `moveto player`. Look for a book on ground.
- When you return to ask Urag if he has translated the writings, he may not give you any reward.
- If you have started [Skyrim:Destroy the Dark Brotherhood!](Skyrim_Destroy_the_Dark_Brotherhood!.md) and cannot find the quest marker for Shalidor's insights on either Skyrim or Solstheim the questline is permanently broken without console commands (because the book is on the nonexistent ship Katariah). - [![On PC](https://images.uesp.net/d/d7/Computer.svg)](https://www.wikipedia.org/wiki/PC_game) Type `sqv mgr21` into the console. Search for Ref 'Book'-> (x) where x is the Book Ref ID code. Type `Prid x` and press enter. Then type `moveto player`. The book should fall on the floor in front of you. Remember to do this in a private location because picking up the book will be classified as stealing.
- Urag repeats the lengthy introduction to this quest every time it starts because the tracker variable resets each time. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 1.3.2, fixes this bug.
