[![](https://images.uesp.net/thumb/3/36/SR-flora-Spiky_Grass_2.jpg/200px-SR-flora-Spiky_Grass_2.jpg)](https://en.uesp.net/wiki/File:SR-flora-Spiky_Grass_2.jpg) [](https://en.uesp.net/wiki/File:SR-flora-Spiky_Grass_2.jpg) Spiky Grass (ashen) **Ashen Grass Pod** is an ingredient harvested from Spiky Grass plants unique to [Solstheim](Skyrim_Solstheim.md). The plant looks similar to the [grass pod](Skyrim_Grass_Pod.md) found in mainland [Skyrim](Lore_Skyrim.md), only this type is grey in color and has different alchemical properties.

Guaranteed samples can be found in the following locations:

- 1 in [Raven Rock Mine](Skyrim_Raven_Rock_Mine.md), on an alchemy table
- 1 in [Severin Manor](Skyrim_Severin_Manor.md) ([Raven Rock](Skyrim_Raven_Rock.md)), on a shelf next to the alchemy table downstairs
