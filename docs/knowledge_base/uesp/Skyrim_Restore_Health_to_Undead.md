# Restore Health to Undead

| [](https://en.uesp.net/wiki/File:SR-icon-spell-Heal.png) | Restore Health to Undead |
| --- | --- |
| School | [Restoration](Skyrim_Restoration.md) |
| Type | Restorative |

**Heals the undead target <mag> points per second, but not the living, atronachs or machines.**

**Restore Health to Undead** allows for the spellcaster to heal an [undead](Skyrim_Undead.md) target. The effect is only available in the game as a spell.

## Spells
The following spells have the Restore Health to Undead effect. Following each spell is the specific variant of the effect that it uses:

- [Necromantic Healing](Skyrim_Necromantic_Healing.md): *Restore Health to Undead* (`xx00e8d2`; `DLC1Necromantic Healing`)
- [Heal Undead](Skyrim_Heal_Undead.md): *Restore Health to Undead* (`xx00e8d4`; `DLC1Heal Undead`)
