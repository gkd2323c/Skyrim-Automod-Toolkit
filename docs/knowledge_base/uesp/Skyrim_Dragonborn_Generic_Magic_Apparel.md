# Dragonborn Generic Magic Apparel

This page describes the **Generic Magic Apparel** items in Dragonborn. All of the randomly generated enchanted armor in the DLC are listed on this page, including items found on enemy NPCs and items found in loot.

**See Also**:

- [Dragonborn Items](Skyrim_Dragonborn_Items.md): General information on magic items and how they appear.
- [Dragonborn Armor](Skyrim_Armor.md): All of the basic (unenchanted) armor in the game.
- [Dragonborn Clothing](Skyrim_Clothing.md): All of the basic (unenchanted) clothing and jewelry in the game.
- [Dragonborn Leveled Items](Skyrim_Leveled_Items.md): Leveled magic apparel, generally received as quest rewards.
- [Dragonborn Unique Items](Skyrim_Unique_Items.md#Dragonborn): Unique apparel, available from merchants or received as quest rewards.
- [Dragonborn Artifacts](Skyrim_Artifacts.md): Unusually powerful one-of-a-kind apparel.

[Tempering](Skyrim_Workbench.md) any magic armor requires the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk to be unlocked. The type of ingot (or other material) necessary for the upgrade, as well as which [Smithing](Skyrim_Smithing.md) perk augments the upgrade, is the same as for the base version of the armor. The base version of each armor is highlighted via a link in the armor name; that link leads to the page detailing the base armor.

Unless noted otherwise, all of these items can be disenchanted.
