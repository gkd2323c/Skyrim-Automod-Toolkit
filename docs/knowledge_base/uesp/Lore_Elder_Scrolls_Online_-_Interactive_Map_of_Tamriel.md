# Elder Scrolls Online - Interactive Map of Tamriel

[![](https://images.uesp.net/thumb/1/1d/ON-map-Regions_%28labeled%29.jpg/300px-ON-map-Regions_%28labeled%29.jpg)](https://en.uesp.net/wiki/File:ON-map-Regions_(labeled).jpg) [](https://en.uesp.net/wiki/File:ON-map-Regions_(labeled).jpg) Interactive Map This page is a list of all texts and documents found on the [Zeni Max](https://en.uesp.net/wiki/General:Zeni Max_Online_Studios) [Interactive Map of Tamriel](http://www.elderscrollsonline.com/en-uk/map/tamriel/), released prior to *[Elder Scrolls Online](https://en.uesp.net/wiki/Online:Online)*. It was updated once to include detailed information about [Cyrodiil](https://en.uesp.net/wiki/Online:Cyrodiil), and again for the release of [Lower Craglorn](https://en.uesp.net/wiki/Online:Lower_Craglorn). It was subsequently discontinued and is no longer available on the site. Texts contain information that can be used as a reference in [lore articles](Lore_Main_Page.md).
