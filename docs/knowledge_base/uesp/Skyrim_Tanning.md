## Tanning Transformations
Leather is unusually heavy, meaning that in nearly all cases leather weighs more than the hides/pelts used to create the leather. Therefore, tanning when near your [Carry Weight](Skyrim_Carry_Weight.md) limit can cause you to be over-encumbered. However, further transforming the leather into leather strips always results in a net decrease in weight.

Several pelts -- all varieties of bear pelts as well as Ice Wolf Pelts -- are more valuable originally than after being converted into leather. Even if the leather is further transformed in leather strips, there is still a net decrease in gold value. Confusingly enough, [Deer Pelts](https://en.uesp.net/wiki/Skyrim:Deer_Pelt) cannot be used to create leather.

## Bugs
- Occasionally, when you activate the Tanning Rack, the animation will not start and you will not enter third-person view if you were in first-person view. If this happens, the game will crash as soon as you try to craft anything. <sup>**?**</sup> - If this happens, exit the screen, wait a few seconds, and then try again.
