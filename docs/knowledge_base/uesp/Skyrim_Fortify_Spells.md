*Enchanting description:* **Spells cost 3% less to cast.**

## Enchanting
Only one item uses this effect, but it cannot be disenchanted:

- [Saarthal Amulet](https://en.uesp.net/wiki/Skyrim:Saarthal_Amulet)

## Notes
- The [Diadem of the Savant](Skyrim_Diadem_of_the_Savant.md) and [Archmage's Robes](Skyrim_Archmage%27s_Robes.md) provide an equivalent effect (and are slightly stronger, at a 5% and 15%, respectively, cost reduction for all spells) but do so with a different enchantment that contains all five schools separately. They cannot be disenchanted either.

- This effect is actually just a different version of [Fortify Alteration](Skyrim_Fortify_Alteration.md) rather than fortify all spells.
