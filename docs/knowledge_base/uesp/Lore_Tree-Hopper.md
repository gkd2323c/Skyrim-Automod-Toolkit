# Tree-Hopper

| Tree-Hoppers | |
| --- | --- |
| Type | [Beast](Lore_Bestiary.md) |
| Range | [Valenwood](Lore_Valenwood.md) |

Tree-Hoppers are creatures that inhabit [Valenwood](Lore_Valenwood.md). The [Bosmer](Lore_Bosmer.md) use spells to repel them as part of the process for brewing [beer](https://en.uesp.net/wiki/Lore:Beer) from meat.<sup>[[1]](#cite_note-OTBODMB-1)</sup> The term "tree-hopper," derived from them, was also used to describe peaceful, innocent people.<sup>[[2]](#cite_note-GST-2)</sup>

## Notes
- Tree-Hoppers originally appeared in the unofficial text *[The Dreadful Theft of the Sun's Dusk Ale](https://en.uesp.net/wiki/General:The_Dreadful_Theft_of_the_Sun%27s_Dusk_Ale)*.<sup>[[UOL 1]](#cite_note-UOLTD-3)</sup>
- In real life, [treehoppers](https://www.wikipedia.org/wiki/Treehopper) are a species of insect.
