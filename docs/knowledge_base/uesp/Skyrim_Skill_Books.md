# Skill Books


- When you pick up a skill book, you automatically start to read it, and therefore immediately obtain the skill bonus. As with all books, you can read skill books without it being considered [theft](Skyrim_Crime.md); theft only occurs if you select to "Take" an owned book while reading it. If you want to save a skill book for later, you can ask your [follower](Skyrim_Followers.md) to take or steal it for you.
- Reading a new skill book awards sufficient experience to raise the skill to the next level (though see [bugs](#Bugs)). Therefore, to gain maximum experience, avoid reading skill books when your skill is about to level up.
- As with skill trainers, experience gained from skill books is increased with standing stone effects and other xp modifiers making it possible to earn over 2 levels worth of xp (however, it is not possible to level up a skill 3 times with one book) at a time while using scholar’s insight, an appropriate guardian stone, any of the bed rested bonuses, and the lovers stone on the Aetherial Crown. This will allow for “bonus” levels to be earned if xp bar growth is tracked and the appropriate skill is used in its normal fashion in instances where the xp bar progression reaches or nearly reaches the end of that levels bar but hasn’t moved to the next level. Performing this tedium when skills are level 85 or so can nearly max a skill after 5 skill books.
- Reading another copy of a skill book that you have already read will not award any additional experience in that skill.
- Generally speaking, it's more efficient to read skill books when you are at a fairly high level in that skill, as the time and effort required to increase skills at high levels is much greater. For example, leveling Archery from level 15 to level 16 can be achieved with just a few attacks; thus, using a skill book at this point is a waste. In contrast, leveling Archery from level 60 to 61 requires significantly more attacks, so reading a skill book at this point is more efficient.
- Making a skill Legendary does not reset skill books, so after a skill has been made Legendary, you will not receive any experience for that skill from any of its skill books that have already been read.
- Each newly read skill book awards two skill level increases when the [Scholar's Insight](https://en.uesp.net/wiki/Skyrim:Scholar%27s_Insight) ability obtained from the *[Dragonborn](Skyrim_Dragonborn.md)* quest [The Winds of Change](Skyrim_Black_Book__The_Winds_of_Change_(quest).md) is active.

## Locations
Skill book locations are listed in the following table. With the exception of skill books found on the remains of [Seekers](Skyrim_Seeker.md) in [Apocrypha](Skyrim_Apocrypha.md) (which will never include unread skill books), they are never found in random loot, and they are never part of a merchant's inventory. Skill books can be easily distinguished from most other books by their value, which is usually at least 50 gold, substantially higher than the value of most other books. They can be sold to any merchants who trade in books, including spell and general goods merchants.

## Achievements
The [achievement](Skyrim_Achievements.md) related to skill books is:

- [![SR-achievement-Reader.png](https://images.uesp.net/thumb/6/6d/SR-achievement-Reader.png/30px-SR-achievement-Reader.png)](https://en.uesp.net/wiki/File:SR-achievement-Reader.png) **Reader** (20 points/Bronze) — Read 50 Skill Books - To unlock this achievement, 50 different skill books must be read out of the 90 available.
- Books count towards this achievement even if they are read after the related skill has reached 100. In this case, no message appears on screen informing you that you have read a new skill book, but the game's counter is still increased.
- You can track your progress towards the achievement using the "Skill Books Read" statistic on the General page of your journal's General Stats.

## Bugs
- On rare occasions, reading a skill book will report the skill being leveled, but not actually increase the skill's level. Being close to leveling the skill may increase the likelihood of this happening. - Save your game and check the skill's value before reading any books. If it doesn't increase after reading, reload the save and increase the level of the skill before reading the book.
- Some skill books will not appear unless you have already found a different copy of that skill book, and some skill books will not appear if you have already found a different copy of that skill book. <sup>**?**</sup>
