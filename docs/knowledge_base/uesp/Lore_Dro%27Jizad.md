# Dro'Jizad

| Dro'Jizad | | | |
| --- | --- | --- | --- |

| Resided in | [Elsweyr](Lore_Elsweyr.md) | | |

Dro'Jizad was a well-known [Khajiit](Lore_Khajiit.md)<sup>[[nb 1]](#intnote_nb1)</sup> [prophet](Lore_Sight.md) from [Elsweyr](Lore_Elsweyr.md) of some renown, known for a number of prophecies. His predictions were listed alongside the [Nerevarine Prophecy](Lore_Nerevarine_Prophecy.md) and secrets of the [Elder Scrolls](Lore_Elder_Scrolls.md). He lived at some point before the late [Third Era](Lore_Third_Era.md). As of that time, his prophecies were still kept secret and were not yet revealed to the public.<sup>[[1]](#cite_note-THOR-1)</sup>
