## Notes
- Boneshaver has previously appeared in *[Blades](https://en.uesp.net/wiki/Blades:Boneshaver)*. For more information, see the [lore article](Lore_Boneshaver.md).
- Boneshaver has no usage cost, so its charges will never drain.
