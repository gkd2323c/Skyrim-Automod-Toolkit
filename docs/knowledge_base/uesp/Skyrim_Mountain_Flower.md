# Mountain Flower

**Mountain Flowers** can give one of four ingredients:

- [Blue Mountain Flower](Skyrim_Blue_Mountain_Flower.md), a blue version of the flower
- [Purple Mountain Flower](Skyrim_Purple_Mountain_Flower.md), a purple version of the flower
- [Red Mountain Flower](Skyrim_Red_Mountain_Flower.md), a red version of the flower
- [Yellow Mountain Flower](Skyrim_Yellow_Mountain_Flower.md), a yellow version of the flower
