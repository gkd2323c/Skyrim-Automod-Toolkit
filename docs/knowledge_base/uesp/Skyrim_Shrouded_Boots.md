[![](https://images.uesp.net/thumb/5/51/SR-item-Shrouded_Boots.jpg/200px-SR-item-Shrouded_Boots.jpg)](https://en.uesp.net/wiki/File:SR-item-Shrouded_Boots.jpg) [](https://en.uesp.net/wiki/File:SR-item-Shrouded_Boots.jpg) Shrouded Boots Shrouded Boots are the boots for a set of armor used by the [Dark Brotherhood](Skyrim_Dark_Brotherhood.md). Alongside these shoes, this set of armor also includes [armor](Skyrim_Shrouded_Armor.md), a [cowl](Skyrim_Shrouded_Cowl.md), and [gloves](Skyrim_Shrouded_Gloves.md). These boots [muffle](Skyrim_Muffle_(effect).md) movement by half a point, making silent movement easier. [Tempering](Skyrim_Workbench.md) Shrouded Boots requires a piece of [leather](Skyrim_Leather.md) and the [Arcane Blacksmith](Skyrim_Arcane_Blacksmith.md) perk.
