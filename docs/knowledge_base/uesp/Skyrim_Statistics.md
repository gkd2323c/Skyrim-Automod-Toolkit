# Statistics


## General
| Statistic | Notes |
| --- | --- |
| Mauls | The number of NPCs or creatures killed with a melee attack while in the form of a [werewolf](Skyrim_Lycanthropy.md#Beast_Form) or [Vampire Lord](Skyrim_Vampire_Lord.md). Special finishing moves performed on a target are also counted. |
| Werewolf Transformations | The number of times you have used the [Beast Form](Skyrim_Lycanthropy.md#Beast_Form) power. |
| Days as a Werewolf | The number of days that have passed since becoming afflicted with [Lycanthropy](Skyrim_Lycanthropy.md). |
| Necks Bitten | The number of times you have [fed upon](Skyrim_Vampirism.md#Feeding) an NPC as a [vampire](Skyrim_Vampirism.md). |
| Days as a Vampire | The number of days that have passed since becoming afflicted with [Sanguinare Vampiris](Skyrim_Sanguinare_Vampiris.md). |
| Locations Discovered | The number of [locations](Skyrim_Places.md) you have discovered. Only locations that are marked on the game map count towards this statistic. |
| Dungeons Cleared | The number of dungeons [cleared](Skyrim_Dungeons.md#Clearing). A dungeon is typically cleared once the boss has been defeated. There are a total of 186 locations that can be cleared, however once a location [respawns](Skyrim_Respawning.md), it may be cleared again. This means the statistic displayed may go well beyond the dungeon limit. |
| Days Passed | The number of days passed since the game started. The game [starts](Skyrim_Unbound.md) on [Morndas, the 17th of Last Seed](Lore_Calendar.md#Skyrim_Calendar), 4E 201. |
| Hours Slept | The number of hours [slept](Skyrim_Sleeping.md). You can only sleep in an unowned bed and if there are no enemies nearby. |
| Hours Waiting | The number of hours [waited](Skyrim_Sleeping.md#Waiting). You can wait almost anywhere, but only if there are no enemies nearby. |
| Standing Stones Found | The number of [Standing Stones](Skyrim_Standing_Stone.md) discovered. There are 13 Standing Stones throughout Skyrim, however visiting a previously discovered standing stone will add to this statistic, making it possible to go well over the original 13. [All-Maker Stones](Skyrim_All-Maker_Stones.md) included in [Dragonborn](Skyrim_Dragonborn.md) are not counted towards this total. |
| Gold Found | The grand total of [gold](Skyrim_Gold.md#Currency) coins found. |
| Most Gold Carried | The most gold carried at any one time. |
| Chests Looted | The number of [chests](Skyrim_Containers.md#Chest) opened. |
| Skill Increases | The number of times a skill increases. There are 18 [skills](Skyrim_Skills.md) in total, each of which can level up to 100. With the [1.9 patch](Skyrim_Patch.md#Version_1.9), which includes a [skill reset](Skyrim_Skills.md#Legendary_Skills) option, there is no limit to how many times a skill can be increased. |
| Skill Books Read | The number of [skill books](Skyrim_Skill_Books.md) you have read and gained a skill increase from. There are 5 different skill books for each skill and a total of 90 books. |
| Food Eaten | The number of [food](Skyrim_Food.md) items consumed. |
| Training Sessions | The number of times you have received training from a [skill trainer](Skyrim_Trainers.md). There are 59 trainers in total, including add-ons. |
| Books Read | The number of [books](Skyrim_Books.md) you have read. Simply activating a book will count as reading it. Skill books are not counted towards this statistic, but spell tomes are counted (even if the spell is already learned without using the spell tome). |
| Horses Owned | The number of [horses](Skyrim_Horses.md) you have obtained. Horses are available for purchase at any [stable](Skyrim_Stables.md) for 1000 gold, in the five major holds and the three Hearthfire houses. [Shadowmere](https://en.uesp.net/wiki/Skyrim:Shadowmere) is also counted if obtained, however [Arvak](https://en.uesp.net/wiki/Skyrim:Arvak) and [Frost](https://en.uesp.net/wiki/Skyrim:Frost) are not. From the Creation Club, the horse from the [Farming](Skyrim_Farming.md) creation counts, as does the Reindeer from the [Saturalia Holiday Pack](Skyrim_Saturalia_Holiday_Pack.md) |
| Houses Owned | The number of [houses](Skyrim_Houses.md) you have bought. Only the 5 purchasable houses in the main cities are counted. |
| Stores Invested In | The number of [merchants](Skyrim_Merchants.md) who you have invested gold into. Requires the [Investor](Skyrim_Investor.md) perk in the [Speech](Skyrim_Speech.md) skill tree. |
| Barters | The act of one transaction, either buying or selling, with a [merchant](Skyrim_Merchants.md) counts as a barter. |
| Persuasions | The number of times a successful [persuasion](Skyrim_Persuasion.md) is performed. |
| Bribes | The number of times a successful [bribe](https://en.uesp.net/wiki/Skyrim:Bribe) is performed. |
| Intimidations | The number of times a successful [intimidation](https://en.uesp.net/wiki/Skyrim:Intimidate) is performed. |
| Diseases Contracted | The total number of [diseases](Skyrim_Disease.md) you have contracted. There are 8 different diseases, all of which can be contracted multiple times. |

## Quest
| Statistic | Notes |
| --- | --- |
| Dragonborn Quests Completed <sup>[DB](Skyrim_Dragonborn.md)</sup> | The number of quests completed in the [Dragonborn](Skyrim_Dragonborn.md) [main questline](Skyrim_Dragonborn_Quests.md). 7 in total.<sup>[*verification needed — see [talk page](https://en.uesp.net/wiki/Skyrim:talk_Statistics)*]</sup> |
| Dawnguard Quests Completed <sup>[DG](Skyrim_Dawnguard.md)</sup> | The number of quests completed for the [Dawnguard](Skyrim_Dawnguard_(faction).md#Quests) or Castle Volkihar. Maximum is 21 for Volkihar, 20 for the Dawnguard. Each type of radiant quest can only be counted once, and not all radiant quests are counted. Impatience of a Saint is also counted. |
| Quests Completed | The grand total of [all quests](Skyrim_Quests.md) completed. |
| Misc Objectives Completed | The number of [miscellaneous quests](Skyrim_Miscellaneous_Quests.md) completed. These are not classed as full quests and many are repeatable. |
| Main Quests Completed | The number of [main quests](Skyrim_Main_Quest.md) completed. 19 in total, including two optional quests. |
| Side Quests Completed | The number of [side quests](Skyrim_Side_Quests.md) completed. No limit, since some are infinitely repeatable. |
| The Companions Quests Completed | The number of [primary](Skyrim_Companions.md#Primary_Quests) and two [radiant quests](Skyrim_The_Companions.md#Radiant_Quests) completed for the [Companions](Skyrim_Companions.md). 8 in total. This number can be higher due to a [bug](Skyrim_The_Companions.md#Bugs). |
| College of Winterhold Quests Completed | The number of [primary](Skyrim_College_of_Winterhold_(faction).md#Primary_Quests) and [side quests](Skyrim_College_of_Winterhold_(faction).md#Optional_Quests) completed for the [College of Winterhold](Skyrim_College_of_Winterhold_(faction).md). Maximum is 19. |
| Thieves' Guild Quests Completed | The number of [primary](Skyrim_Thieves_Guild_(faction).md#Primary_Quests) quests and [special jobs](Skyrim_Thieves_Guild_(faction).md#Special_Job_Quests) completed for the [Thieves' Guild](Skyrim_Thieves_Guild_(faction).md). Maximum is 24. Each type of radiant quest can only be counted once. |
| The Dark Brotherhood Quests Completed | The number of [primary quests](Skyrim_Dark_Brotherhood.md#Primary_Quests) and [contracts](Skyrim_Dark_Brotherhood.md#Contract_Quests) completed for the [Dark Brotherhood](Skyrim_Dark_Brotherhood.md). Maximum permanent value is 27, but may be temporarily increased to 28 after completing a radiant quest. Only 2 possible if you [destroy the Dark Brotherhood](Skyrim_Destroy_the_Dark_Brotherhood!.md). Repeatable [radiant quests](Skyrim_Dark_Brotherhood.md#Radiant_Quests) are not included. |
| Civil War Quests Completed | The number of quests completed for either side of the [civil war](Skyrim_Civil_War.md) questline. 9 in total, but the number may change if you have completed Season Unending. |
| Daedric Quests Completed | The number of [Daedric quests](Skyrim_Daedric_Quests.md) completed. There are 15 in total. Daedric quests are not considered side quests. |
| Questlines Completed | Questlines are typically classed as a series of quests for a joinable [faction](Skyrim_Factions.md), but it also includes the main questline of Skyrim. 6 in total. |

## Combat
| Statistic | Notes |
| --- | --- |
| People Killed | The number of NPCs, both hostile and non-hostile, you have killed. |
| Animals Killed | The number of [animals](Skyrim_Animals.md) you have killed. Also includes all types of [conjured familiars](Skyrim_Conjure_Familiar.md). |
| Creatures Killed | The number of [creatures](Skyrim_Monsters.md) you have killed. [Animals](Skyrim_Animals.md) or [undead](Skyrim_Undead.md) are not included. |
| Undead Killed | The number of [undead](Skyrim_Undead.md) you have killed. Includes [draugr](Skyrim_Draugr.md), [ghosts](Skyrim_Ghost.md), [skeletons](Skyrim_Skeleton.md) and [vampires](Skyrim_Vampire.md) and all types of [conjured](Skyrim_Conjuration.md#Spells) undead, such as [bonemen](https://en.uesp.net/wiki/Skyrim:Conjure_Boneman). |
| Daedra Killed | The number of Daedra you have killed. Includes [Dremora](Skyrim_Dremora.md), elemental [atronachs](Skyrim_Daedra.md) and all types of summoned Daedra. |
| Automatons Killed | The number of [Dwarven automatons](Skyrim_Dwarven_Automatons.md) you have defeated. Includes [Dwarven spiders](Skyrim_Dwarven_Spider.md), [spheres](Skyrim_Dwarven_Sphere.md), [centurions](Skyrim_Dwarven_Centurion.md) and [ballistae](Skyrim_Dwarven_Ballista.md). |
| Favorite Weapon | The [weapon](Skyrim_Weapons.md) that is most often used. Limited to melee weapons or bows. |
| Critical Strikes | The number of critical strikes performed. A critical strike is a special attack that, if triggered, deals more damage than a normal blow. It can be performed in both regular combat or in stealth, and can only be achieved with melee weapons or bows. A successful [sneak attack](Skyrim_Sneak.md#Sneak_Attacks) will always result in a critical strike. |
| Sneak Attacks | The number of [sneak attacks](Skyrim_Sneak.md#Sneak_Attacks) performed. A sneak attack is the act of striking a target while remaining [undetected](Skyrim_Sneak.md#Remaining_Undetected). |
| Backstabs | The number of backstabs performed. A backstab is the act of performing a sneak attack with a one-handed weapon on a target facing away from you. It does not require the [Backstab](Skyrim_Sneak.md#Backstab) perk to trigger. |
| Weapons Disarmed | The number of NPCs you have successfully disarmed. Disarms can be achieved with either the [Disarm](Skyrim_Disarm.md) shout directed at a target or by performing a [power bash](Skyrim_Block.md#Bashing) attack with a shield, weapon or torch and the [Disarming Bash](https://en.uesp.net/wiki/Skyrim:Disarming_Bash) perk. |
| Brawls Won | The number of [brawls](Skyrim_Brawl.md) you have won against [various NPCs](Skyrim_Fight!_Fight!.md). |
| Bunnies Slaughtered | The number of [rabbits](Skyrim_Rabbit.md) you have killed. |

## Magic
| Statistic | Notes |
| --- | --- |
| Spells Learned | The number of [spells](Skyrim_Spells.md) you have learned. A spell tome must be read in order to learn a spell.<sup>[*verification needed — see [talk page](https://en.uesp.net/wiki/Skyrim:talk_Statistics)*]</sup> |
| Favorite Spell | The spell that is most often used. |
| Favorite School | The [school of magic](Skyrim_Magic_Overview.md) that is most often used. The five schools are [Alteration](Skyrim_Alteration.md), [Conjuration](Skyrim_Conjuration.md), [Destruction](Skyrim_Destruction.md), [Illusion](Skyrim_Illusion.md), and [Restoration](Skyrim_Restoration.md). |
| Dragon Souls Collected | Souls collected from slain [dragons](Skyrim_Dragon.md). Souls gain from Miraak in Dragonborn are not counted.<sup>[*verification needed — see [talk page](https://en.uesp.net/wiki/Skyrim:talk_Statistics)*]</sup> |
| Words of Power Learned | The number of Words of Power learned. A Word of Power is defined as one of the three words that make up a full [Dragon Shout](Skyrim_Dragon_Shouts.md). They can be learned from either [word walls](Skyrim_Word_Wall.md) or individuals such as the [Greybeards](Skyrim_Greybeards.md) or [Durnehviir](Skyrim_Durnehviir_(dragon).md). |
| Words of Power Unlocked | The number of Words of Power unlocked. A Word of Power can only be unlocked when a Dragon Soul is spent to unlock it. |
| Shouts Learned | The number of full [Dragon Shouts](Skyrim_Dragon_Shouts.md) learned. A full shout is defined by learning all three Words of Power of a shout. |
| Shouts Mastered | The number of full Dragon Shouts unlocked. Shout mastery is defined by learning and unlocking all three Words of Power of a shout. |
| Times Shouted | The number of times you have used a Dragon Shout. Using any one of the three available levels of a shout counts as one instance. |
| Favorite Shout | The shout that is most often used. |

## Crafting
| Statistic | Notes |
| --- | --- |
| Soul Gems Used | The number of [soul gems](Skyrim_Soul_Gems.md) used to recharge an enchanted weapon only. Soul gems consumed via [Enchanting](Skyrim_Enchanting.md) do not count. |
| Souls Trapped | The number of souls trapped within a soul gem. Souls can only be trapped with the [soul trap](Skyrim_Soul_Trap_(spell).md) spell or with a weapon that bears the [soul trap enchantment](Skyrim_Soul_Trap_(effect).md). |
| Magic Items Made | The number of items [enchanted](Skyrim_Enchanting.md) at an [arcane enchanter](Skyrim_Arcane_Enchanter.md). |
| Weapons Improved | The number of weapons [tempered](Skyrim_Smithing.md#Improving_Items) at a [grindstone](Skyrim_Grindstone.md). |
| Weapons Made | The number of [weapons](Skyrim_Weapons.md) created at a [forge](Skyrim_Forge.md). |
| Armor Improved | The number of [armor](Skyrim_Armor.md) pieces tempered at a [workbench](Skyrim_Workbench.md). |
| Armor Made | The number of armor pieces created at a forge. |
| Potions Mixed | The number of [potions](Skyrim_Potions.md) made at an [alchemy lab](Skyrim_Alchemy_Lab.md). Strangely, food you cook will also increase this stat. |
| Potions Used | The number of potions consumed. |
| Poisons Mixed | The number of [poisons](Skyrim_Poisons.md) made at an alchemy lab. |
| Poisons Used | The number of times a poison is applied to a weapon. Poisons placed in the pockets of NPCs with the [Poisoned](Skyrim_Poisoned.md) perk do not count towards this statistic. |
| Ingredients Harvested | The number of alchemy [ingredients](Skyrim_Ingredients.md) harvested from plants or collected from certain [passive creatures](Skyrim_Passive_Creatures.md).<sup>[*verification needed — see [talk page](https://en.uesp.net/wiki/Skyrim:talk_Statistics)*]</sup> |
| Ingredients Eaten | The number of alchemy ingredients eaten. |
| Nirnroots Found | The number of [nirnroot](https://en.uesp.net/wiki/Skyrim:Nirnroot) plants harvested. They can only be found growing in the wild, typically near water. [Crimson nirnroot](https://en.uesp.net/wiki/Skyrim:Crimson_Nirnroot) does not count towards this statistic. |
| Wings Plucked | The number of wings plucked from certain [winged insects](Skyrim_Passive_Creatures.md#Insects), such as butterflies. Plucking wings always yields two samples, thus this statistic will be an even number. |

## Crime
| Statistic | Notes |
| --- | --- |
| Total Lifetime Bounty | The grand total of all [bounties](Skyrim_Crime.md#Bounties), paid and unpaid, placed upon you. The [Dark Brotherhood](Skyrim_Dark_Brotherhood.md) quest [To Kill an Empire](Skyrim_To_Kill_an_Empire.md) incurs an unavoidable 1500 gold bounty. |
| <Hold Name> Bounty | The bounty that you have accrued in a specific [hold](Skyrim_Holds.md). This statistic will only appear if you have incurred a bounty and multiple holds can be listed. |
| Tribal Orcs Bounty | The bounty that you have accrued within an [Orc stronghold](https://en.uesp.net/wiki/Skyrim:Orc_Strongholds). This statistic will only appear if you have incurred a bounty. All 4 strongholds track their bounties collectively. |
| Largest Bounty | The largest total bounty you have ever accrued at any one time. The largest possible bounty for a single crime is murder or transforming into a [werewolf](Skyrim_Lycanthropy.md) or [Vampire Lord](Skyrim_Vampire_Lord.md), which yields a 1000 gold bounty. |
| Locks Picked | The number of locked doors or containers successfully unlocked via [lockpicking](Skyrim_Lockpicking.md). |
| Pockets Picked | The number of separate instances of performing a [pickpocket](Skyrim_Pickpocket.md) on an NPC. |
| Items Pickpocketed | The number of items you have pickpocketed from an NPC's inventory. |
| Times Jailed | The number of times you have been [sent to jail](Skyrim_Crime.md#Going_to_Jail). |
| Days Jailed | The number of days you spend in jail [serving your sentence](Skyrim_Crime.md#Serving_Your_Sentence). |
| Fines Paid | The total amount of gold paid for any [bounties](Skyrim_Crime.md#Paying_Bounty) you incur. |
| Jail Escapes | The number of times you have [escaped](Skyrim_Crime.md#Escaping_Jail) jail, including [The Chill](https://en.uesp.net/wiki/Skyrim:The_Chill). Escaping from [Cidhna Mine](Skyrim_Cidhna_Mine.md) during [No One Escapes Cidhna Mine](Skyrim_No_One_Escapes_Cidhna_Mine.md) quest does not count towards this statistic. |
| Items Stolen | The number of owned items you have added to your inventory. Can later result in being attacked by a group of [hired thugs](Skyrim_Hired_Thug.md). |
| Assaults | The number of separate instances of performing an unprovoked attack upon a non-hostile NPC. Typically results in a bounty if witnessed. |
| Murders | The number of non-hostile NPC deaths you are responsible for. [Followers](Skyrim_Followers.md) and [summoned creatures](https://en.uesp.net/wiki/Skyrim:Conjure#Spells) who kill on your behalf also count towards this statistic. |
| Horses Stolen | The number of owned [horses](Skyrim_Horses.md) mounted. The horse need not actually be stolen to increase this statistic. |
| Trespasses | The number of times you have trespassed. A trespass occurs when entering a locked property, such as a shop, after it has closed. |

## Bugs
- Quitting a Thieves Guild special job will reduce the "Thieves' Guild Quests Completed" stat by 1.
- Upon starting a new game the "Most Gold Carried" stat may be set to 140. This won't affect gameplay and the stat will update regularly if the player has >140 [Gold](Skyrim_Gold.md) at any point <sup>**?**</sup>
