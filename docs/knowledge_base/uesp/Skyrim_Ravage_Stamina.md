*Alchemy description:* **Concentrated poison damages maximum stamina by <mag> points.**
This effect is a variant of [Damage Stamina](Skyrim_Damage_Stamina.md). It causes a total of <mag> points of damage, instantaneously.

As with the other Damage Stamina variant, [Lingering Damage Stamina](Skyrim_Lingering_Damage_Stamina.md), it is only available from poisons.

## Notes
- Ravage Stamina poisons are a good choice against melee opponents since they won't be able to block as effectively or execute as many power attacks.
- Because Ravage Stamina has a relatively short duration, it is best used for finishing a fight with a melee opponent. If used at the start of a fight, it will typically wear off before the target is dead. However, it *can* stack multiple times if applied quickly enough.

## Bugs
- The potion has a duration, but the description only says "Concentrated poison damages maximum Stamina by x points."- ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.0, addresses this issue. The potion's description now specifies how long the duration is.
- The description "Concentrated poison drains Stamina by x points per second for x seconds" doesn't match the potion's effect. - ![PC Only](https://images.uesp.net/d/d7/Computer.svg)
The [Unofficial Skyrim Patch](https://en.uesp.net/wiki/Skyrim_Mod:Unofficial_Skyrim_Patch), version 2.0.9, addresses this issue. The description now says "Concentrated poison temporarily reduces maximum and current Stamina by x points for x seconds."
