The **Skyrim Game Manual** is the instruction manual included with *[The Elder Scrolls V: Skyrim](Skyrim_Skyrim.md)*. The manual gives basic instructions to gameplay mechanics. It can be downloaded [here](https://manuals.bethsoft.com/).

## Notes
- The Legendary Edition version of the manual explains how to start each of the [Add-Ons](Skyrim_Add-ons.md) before the Table of Contents.
- The [Special Edition](Skyrim_Special_Edition.md) version of the manual is a two-page pamphlet only providing gameplay controls and technical information.
- Screenshots in the manual shows the [player-character](Skyrim_Prisoner.md) is a [Nord](Skyrim_Nord.md) named [Bendu Olo](https://en.uesp.net/wiki/Lore:Bendu_Olo).
