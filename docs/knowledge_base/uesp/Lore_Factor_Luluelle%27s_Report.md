Factor Luluelle's Report **by [Lacherie Luluelle](Lore_Books_by_Author.md#Lacherie_Luluelle)** *A half-written report regarding the cargo aboard the Whispering Oyster* *
---
Arrived from [Sentinel](Lore_Sentinel.md) on the "Whispering Oyster":

—Three Barrels [Pomegranate](Lore_Pomegranate.md) [Wine](Lore_Wine.md)

—Four Crates [Mountain Jerky](Lore_Mountain_Jerky.md); one crate damaged, spoilage significant

—Two Crates —

Oh, what's the point? This is the only shipment we've had in ten days, and its value wouldn't add up to a score of [gold drakes](Lore_Currency.md), even if we had a caravel to trans-ship it someplace where we could sell it. Everything that floats has been "requisitioned" for the [war](Lore_Three_Banners_War.md), and we couldn't get naval stores even if we had a ship—you can't find rope, pitch, timber, or sailcloth for any amount of money. It's all "Arms for [the Covenant](Lore_Daggerfall_Covenant.md)!" now. Let's face it: if this war doesn't end soon, we're shipwrecked.

Maybe I'd be happier if my son Durant was still around, but he ran off and joined the First Menevian Scouts, and the house is empty without him. [Daedra](Lore_Daedra.md) take it—I'm opening one of these barrels to test the contents. If you want to join me, I'll be in the second empty warehouse on the left.

— Lacherie
