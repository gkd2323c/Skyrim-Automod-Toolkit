Bandit's Note on Hidden Treasure **by [Seducers Bandit](https://en.uesp.net/wiki/Skyrim:Seducers_Bandit)** *A rumor of a valuable treasure*
---
An exotic treasure is said to be hidden in [Crystaldrift Cave](Skyrim_Crystaldrift_Cave.md). Heard it from a coinless trader trying to save his own skin. Going to send a few to see if his story checks out.

If the story is true, we need to be sure the [Saints](https://en.uesp.net/wiki/Skyrim:Saints_and_Seducers_Bandits) don't find it first. Our benefactor will be pleased.
