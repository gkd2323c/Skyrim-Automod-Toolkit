# Telekinesis Effect

| [](https://en.uesp.net/wiki/File:SR-icon-spell-Telekinesis.png) | Telekinesis Effect |
| --- | --- |
| School | [Alteration](Skyrim_Alteration.md) |
| Type | Other |

**Can pull an object to you from a distance. Add it to your inventory or throw it.**

**Telekinesis** allows the caster to pick up, drop and throw items from a distance. When the spell wears off, any item that is being manipulated by the effect will fall to the ground. Thieves may find the spell useful for taking items from far away.

## Spells
Only one spell offers this effect:

- [Telekinesis](Skyrim_Telekinesis.md): *Telekinesis Effect* (`0001a4cb`; `Telekinesis Effect`)

## Notes
- Thrown objects can damage NPCs enough to cause them to become hostile, but it doesn't do enough damage to be useful in combat.
- If you have the dual-casting perk for Alteration, it is possible to control 3-4 objects at once.
- The [Vampire Lord](Skyrim_Vampire_Lord.md)<sup>[DG](Skyrim_Dawnguard.md)</sup> has an advanced but limited version of Telekinesis called Vampiric Grip. They can lift people up and throw them, but that is all it can do.
