## Notes
- You can set up your own [](https://en.uesp.net/wiki/File:SR-mapicon-Camp.png) **[campsite](https://en.uesp.net/wiki/Skyrim:Campsite)** with the [Camping](Skyrim_Camping.md) [Creation](Skyrim_Creation_Club.md) installed. The campsite adds a fast travel marker to your map, although you can only have one campsite set up at a time.
